5. ⚠️ **Type aliases and mappings** - Need to eliminate functional type mappings, keep one file per type

**KEY PRINCIPLE**:
> "ONCE" (all caps) = The global singleton accessor constant  
> "Once" (capital O) = Class name pattern for kernel implementations  
> Environment-specific classes use descriptive names: `NodeJsOnce`, `BrowserOnce`, etc.
