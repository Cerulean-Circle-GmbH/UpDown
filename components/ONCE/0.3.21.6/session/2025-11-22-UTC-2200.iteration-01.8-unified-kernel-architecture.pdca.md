### **Objective**

Create a **unified, environment-agnostic ONCE kernel** where:
1. Single entry point: `ONCE.start()` works identically in Node.js, Browser, Worker, Service Worker
2. Environment detection in Layer 1 (`ONCE.ts`)
3. Shared base class (`AbstractONCEKernel`) for all implementations
4. Consistent P2P terminology throughout (no "client/server")
5. Clean markdown formatting in all documentation

---

### **Problem Statement**

**Issue**: Current architecture has:
1. **Separate implementations**: `DefaultONCE` and `BrowserONCEKernel` share no code
2. **Wrong vocabulary**: "server/client" contradicts P2P paradigm
3. **No environment abstraction**: Manual environment detection everywhere
4. **Inconsistent API**: Different initialization patterns per environment
5. **Wrong layer**: `ONCE.ts` in layer3 (should be layer1 infrastructure)

**Impact**:
- Confusing terminology contradicts architecture
- Hard to add new environments (Worker, ServiceWorker)
- Code duplication between implementations
- No single source of truth for "what is an ONCE kernel"

### **Design**

#### **Architecture Overview**

See: `session/2025-11-22-UTC-2200.once-unified-kernel-architecture.puml`

**Layer 1** (Infrastructure):
```typescript
// layer1/ONCE.ts
export class ONCE {
    private static instance: ONCEKernel | null = null;
    private static environment: string | null = null;
    
    static async start(scenario?: any): Promise<ONCEKernel> {
        const env = this.detectEnvironment();
        const kernel = await this.loadKernel(env);
        await kernel.init(scenario);
        this.instance = kernel;
        return kernel;
    }
    
    private static detectEnvironment(): string {
        // typeof window, process, self, navigator
        // Returns: 'nodejs' | 'browser' | 'worker' | 'service-worker'
    }
    
    private static async loadKernel(env: string): Promise<ONCEKernel> {
        switch(env) {
            case 'nodejs': return new (await import('../layer2/DefaultONCE.js')).DefaultONCE();
            case 'browser': return new (await import('../layer2/BrowserONCEKernel.js')).BrowserONCEKernel();
            // ... etc
        }
    }
    
    static getInstance(): ONCEKernel | null {
        return this.instance;
    }
}
```
