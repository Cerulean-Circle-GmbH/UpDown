## **📝 APPENDIX: Current Architecture State**

### **✅ COMPLETED (Iterations 1.9, 1.10, 1.11)**:

**Unified Kernel Architecture**:
```
Layer 1: Once.ts (Environment Detection)
    ├─ EnvironmentType enum
    ├─ DefaultEnvironmentInfo factory
    └─ Dynamic kernel loading

Layer 2: Kernel Implementations
    ├─ DefaultOnceKernel (abstract base)
    ├─ NodeJsOnce (server)
    └─ BrowserOnce (browser)

Layer 3: Interfaces & Models
    ├─ OnceKernel interface
    ├─ Model base interface
    ├─ Scenario<T extends Model>
    └─ Reference<T> pattern

Layer 4: (Future) Services
Layer 5: Entry Points (ONCECLI.ts)
```

**Naming Conventions**:
- `ONCE` = Global singleton accessor
- `Once` = Class name pattern
- `Reference<T>` = Nullable type pattern
- `...Once` = Environment-specific kernels

**Routes**:
- `/` = Server status
- `/once` = Minimal kernel bootstrap
- `/onceCommunicationLog` = P2P demo
- `/demo` = Demo hub (needs update)
