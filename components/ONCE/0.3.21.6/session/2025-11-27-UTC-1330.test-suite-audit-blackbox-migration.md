**Date**: 2025-11-27 UTC 13:30

**Purpose**: Identify and deprecate white-box tests, design proper black-box regression tests

---

## **🎯 WEB4 TESTING PRINCIPLE**

> **"It is 100x cheaper to write correct tests than to fix them."**

**Black-Box Testing Requirements**:
1. Test **WHAT** the system does, NOT **HOW** it does it
2. Tests should survive internal refactoring
3. Test through public APIs only (CLI, HTTP, WebSocket)
4. No access to internal state (`component.model`, private methods)
5. Use observable behavior (files created, HTTP responses, console output)
6. Enable true regression testing

**White-Box Anti-Patterns** (TO BE DEPRECATED):
- `new NodeJsOnce()` - Direct instantiation
- `component.model.something` - Internal state access
- `component.privateMethod()` - Private method calls
- `expect(component).toBeInstanceOf()` - Implementation details
- Testing implementation details that could change

#### **Test 1: ONCE Kernel Lifecycle** (Replaces `once.test.ts`)

```typescript
/**
 * ONCE Kernel Lifecycle - Black-Box Regression Test
 * Tests observable kernel behavior through CLI and HTTP
 */
describe('ONCE Kernel Lifecycle (Black-Box)', () => {
  it('should start kernel via CLI and respond to health check', async () => {
    const script = `
cd ${componentRoot} && \\
./once startServer > /tmp/once-test.log 2>&1 & \\
ONCE_PID=$! && \\
sleep 2 && \\
curl -s http://localhost:42777/health && \\
kill $ONCE_PID 2>/dev/null && \\
wait $ONCE_PID 2>/dev/null || true
    `;
    
    const { stdout } = await execAsync(script);
    
    // Observable behavior: HTTP response contains expected state
    expect(stdout).toContain('running');
    expect(stdout).toContain('ONCE');
  });

  it('should create scenario file on startup', async () => {
    const script = `
cd ${componentRoot} && \\
rm -rf scenarios/box/fritz/*/ONCE/* && \\
./once startServer > /tmp/once-test.log 2>&1 & \\
ONCE_PID=$! && \\
sleep 2 && \\
SCENARIO_COUNT=$(find scenarios/ -name "*.scenario.json" | wc -l) && \\
kill $ONCE_PID 2>/dev/null && \\
echo $SCENARIO_COUNT
    `;
    
    const { stdout } = await execAsync(script);
    
    // Observable behavior: Scenario file created
    expect(parseInt(stdout.trim())).toBeGreaterThan(0);
  });
  
  it('should shut down gracefully via stopServer', async () => {
    const script = `
cd ${componentRoot} && \\
./once startServer > /tmp/once-test.log 2>&1 & \\
ONCE_PID=$! && \\
sleep 2 && \\
./once stopServer && \\
sleep 1 && \\
! curl -s http://localhost:42777/health 2>/dev/null && \\
echo "SHUTDOWN_SUCCESS"
    `;
    
    const { stdout } = await execAsync(script);
    
    // Observable behavior: Server no longer responds
    expect(stdout).toContain('SHUTDOWN_SUCCESS');
  });
});
```

/**
 * P2P State Transfer - Black-Box Regression Test
 * Tests protocol-less scenario transfer between peers
 */
describe('P2P State Transfer (Black-Box)', () => {
  it('should sync scenarios between primary and client peers', async () => {
    const script = `
cd ${componentRoot} && \\
rm -rf scenarios/box/fritz/*/ONCE/* && \\
./once startServer > /tmp/primary.log 2>&1 & \\
PRIMARY_PID=$! && \\
sleep 2 && \\
PRIMARY_UUID=$(curl -s http://localhost:42777/health | jq -r '.uuid') && \\
./once startServer > /tmp/client.log 2>&1 & \\
CLIENT_PID=$! && \\
sleep 3 && \\
SCENARIO_COUNT=$(find scenarios/ -name "*.scenario.json" | wc -l) && \\
kill $PRIMARY_PID $CLIENT_PID 2>/dev/null && \\
echo $SCENARIO_COUNT
    `;
    
    const { stdout } = await execAsync(script);
    
    // Observable behavior: Multiple scenarios exist (primary + client)
    expect(parseInt(stdout.trim())).toBeGreaterThanOrEqual(2);
  });

  it('should broadcast scenario updates via WebSocket', async () => {
    const script = `
cd ${componentRoot} && \\
./once startServer > /tmp/primary.log 2>&1 & \\
PRIMARY_PID=$! && \\
sleep 2 && \\
# Connect WebSocket client and check for scenario updates
wscat -c ws://localhost:42777 --wait 5 > /tmp/ws-messages.log 2>&1 & \\
WSCAT_PID=$! && \\
./once startServer > /tmp/client.log 2>&1 & \\
CLIENT_PID=$! && \\
sleep 3 && \\
kill $WSCAT_PID $PRIMARY_PID $CLIENT_PID 2>/dev/null && \\
cat /tmp/ws-messages.log
    `;
    
    const { stdout } = await execAsync(script);
    
    // Observable behavior: WebSocket receives scenario JSON
    expect(stdout).toContain('"ior"');
    expect(stdout).toContain('"model"');
  });
});
```

#### **Test 3: Primary Server Housekeeping** (Replaces `primary-housekeeping.test.ts`)

/**
 * Primary Server Housekeeping - Black-Box Regression Test
 * Tests automatic cleanup of stale scenarios
 */
describe('Primary Server Housekeeping (Black-Box)', () => {
  it('should delete shutdown scenarios on startup', async () => {
    const script = `
cd ${componentRoot} && \\
# Create a scenario with state=shutdown
mkdir -p scenarios/box/fritz/test/ONCE/0.3.21.6 && \\
echo '{"ior":{"uuid":"test-shutdown"},"owner":"","model":{"state":"shutdown"}}' > \\
  scenarios/box/fritz/test/ONCE/0.3.21.6/test-shutdown.scenario.json && \\
BEFORE_COUNT=$(find scenarios/ -name "test-shutdown.scenario.json" | wc -l) && \\
./once startServer > /tmp/once-test.log 2>&1 & \\
ONCE_PID=$! && \\
sleep 3 && \\
AFTER_COUNT=$(find scenarios/ -name "test-shutdown.scenario.json" | wc -l) && \\
kill $ONCE_PID 2>/dev/null && \\
echo "BEFORE:$BEFORE_COUNT AFTER:$AFTER_COUNT"
    `;
    
    const { stdout } = await execAsync(script);
    
    // Observable behavior: shutdown scenario deleted
    expect(stdout).toContain('BEFORE:1');
    expect(stdout).toContain('AFTER:0');
  });
  
  it('should verify peer reachability before deleting', async () => {
    const script = `
cd ${componentRoot} && \\
./once startServer > /tmp/primary.log 2>&1 & \\
PRIMARY_PID=$! && \\
sleep 2 && \\
./once startServer > /tmp/client.log 2>&1 & \\
CLIENT_PID=$! && \\
sleep 3 && \\
CLIENT_SCENARIO=$(find scenarios/ -name "*.scenario.json" | grep -v primary | head -1) && \\
# Kill client but leave scenario
kill $CLIENT_PID 2>/dev/null && \\
sleep 1 && \\
# Restart primary - should detect unreachable peer and clean up
kill $PRIMARY_PID 2>/dev/null && \\
sleep 1 && \\
./once startServer > /tmp/primary2.log 2>&1 & \\
PRIMARY_PID=$! && \\
sleep 3 && \\
SCENARIO_EXISTS=$(test -f "$CLIENT_SCENARIO" && echo "YES" || echo "NO") && \\
kill $PRIMARY_PID 2>/dev/null && \\
echo $SCENARIO_EXISTS
    `;
    
    const { stdout } = await execAsync(script);
    
    // Observable behavior: unreachable peer scenario deleted
    expect(stdout.trim()).toBe('NO');
  });
});
```
