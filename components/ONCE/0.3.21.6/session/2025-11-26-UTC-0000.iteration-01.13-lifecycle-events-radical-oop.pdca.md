```
CREATED → INITIALIZING → INITIALIZED → STARTING → RUNNING → STOPPING → STOPPED → SHUTTING_DOWN → SHUTDOWN
Optional: RUNNING ↔ PAUSING → PAUSED → RESUMING → RUNNING
Error: Any state → ERROR
```
