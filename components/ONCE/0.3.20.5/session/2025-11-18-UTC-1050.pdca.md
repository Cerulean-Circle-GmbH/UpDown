**Phase 1: Fix Housekeeping Symlink Handling**

**Design Decision**: Use `fs.lstatSync()` instead of `fs.statSync()` to detect and skip broken symlinks

```typescript
// CURRENT CODE (broken):
const stats = fs.statSync(fullPath);  // Throws ENOENT on broken symlinks

// FIX:
const stats = fs.lstatSync(fullPath);  // Returns symlink info without following
if (stats.isSymbolicLink()) {
  try {
    fs.statSync(fullPath);  // Try to follow - will throw if broken
  } catch (err) {
    if (err.code === 'ENOENT') {
      console.log(`⚠️ Skipping broken symlink: ${fullPath}`);
      continue;  // Skip this file
    }
    throw err;
  }
}
```

## **🔧 DO**

### **Fix Implementation**

**File Modified**: `src/ts/layer2/ServerHierarchyManager.ts`  
**Function**: `performHousekeeping()` → `findScenarios()` nested function  
**Lines**: 1290-1320

**Problem**: Used `fs.statSync()` which throws ENOENT on broken symlinks

**Solution**: Use `fs.lstatSync()` with try-catch to gracefully handle broken symlinks

**Code Change**:
```typescript
// BEFORE (line 1297):
const stat = fs.statSync(fullPath);  // ❌ Throws on broken symlinks

// AFTER (lines 1298-1316):
try {
    // Use lstatSync to detect symlinks without following them
    const stat = fs.lstatSync(fullPath);
    
    if (stat.isDirectory()) {
        results.push(...findScenarios(fullPath));
    } else if (item.endsWith('.scenario.json')) {
        // Include all .scenario.json files (symlinks will be filtered later)
        results.push(fullPath);
    }
} catch (err: any) {
    // Skip files we can't stat (broken symlinks, permission issues, etc.)
    if (err.code === 'ENOENT') {
        console.log(`⚠️ Skipping inaccessible file: ${fullPath}`);
    } else {
        console.warn(`⚠️ Error accessing ${fullPath}:`, err.message);
    }
    continue;
}
```
