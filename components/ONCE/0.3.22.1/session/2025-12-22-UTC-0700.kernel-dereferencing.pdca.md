## P - PLAN

### Why This Is Fundamental

**TRON Feedback**:
> "The kernel knows how to load a scenario from a URL/IOR and the kernel knows how to build and load a component from the scenario and instantiate it so that it is dereferenced."

**Current Gap**:
- File/Folder implement dereferencing locally
- But dereferencing is a UNIVERSAL pattern for ALL UcpComponents
- Kernel should provide centralized methods

### Reference<T> = THREE Stages (from L27)

```
Reference<T> can hold:
├── string         │ IOR string — can LOAD scenario
├── Scenario<M>    │ Scenario JSON — can INSTANTIATE component  
├── T              │ Instance — DEREFERENCED (ready to use)
└── null           │ Not set

Dereferencing Chain:
  IOR string → Scenario → Instance (dereferenced)
       ↓           ↓           ↓
  "ior:..."    { model }   DefaultFile
```

**Code Reference**:

```3062:3132:/Users/Shared/Workspaces/2cuGitHub/UpDown/components/ONCE/0.3.22.1/src/ts/layer2/NodeJsOnce.ts
async componentLoad(componentName: string, version: string): Promise<any> {
  const componentKey = `${componentName}/${version}`;
  
  // 1. Check cache first
  if (this.loadedComponents.has(componentKey)) {
    return this.loadedComponents.get(componentKey);
  }
  
  // 2. Delegate to Web4TSComponent
  const web4ts = await this.getWeb4TSComponent();
  await web4ts.componentStart(componentName, version);  // Ensure built
  
  // 3. Read descriptor for implementationClassName
  const descriptor = await web4ts.componentDescriptorRead(componentName, version);
  const implementationClassName = descriptor?.model?.implementationClassName || componentName;
  
  // 4. Dynamic import
  const componentPath = path.join(this.model.projectRoot, 'components', componentName, version);
  const fileUrl = `file://${componentPath}/dist/ts/layer2/${implementationClassName}.js`;
  const componentModule = await import(fileUrl);
  const ComponentClass = componentModule[implementationClassName];
  
  // 5. Cache and return CLASS
  this.loadedComponents.set(componentKey, ComponentClass);
  return ComponentClass;
}
```

## Detailed PLAN

### KD.1: Research — Identify Where Kernel Methods Should Live (30min)

**Question**: Should methods be on `Once`, `DefaultOnceKernel`, or standalone?

**Analysis**:
- `Once` is the static entry point (singleton access)
- `DefaultOnceKernel` is the abstract base (shared by Browser/Node)
- Both `NodeJsOnce` and `BrowserOnce` extend `DefaultOnceKernel`

**⚠️ CRITICAL: P7 — Async Only in Layer 4**

From Web4 Principles Checklist:
> "Principle 7: Async Only in Layer 4 - Layers 1-3 are synchronous, only Layer 4 uses async"

**Architecture Decision**:

```
Layer 1/2: DefaultOnceKernel (facade)
  │
  │  kernel.dereference(reference)  ← FACADE method
  │        │
  │        ▼
  │  await this.dereferencer.dereference(reference)  ← DELEGATE to Layer 4
  │
Layer 4: Dereferencer (orchestrator)
  │
  │  async dereference(reference): Promise<T>  ← ASYNC lives HERE
  │  async scenarioLoad(ior): Promise<Scenario>
  │  async instantiate(scenario): Promise<T>
```

**DECISION**: 
- Facade on `DefaultOnceKernel` (Layer 2) — awaits and returns
- Async implementation in NEW `Dereferencer` class (Layer 4) — does actual work
- Kernel awaits Layer 4 responses, then calls sync to other layers

### KD.3: Implement Dereferencer.instantiate(scenario) (1.5h)

**Purpose**: Scenario → Instance creation (Layer 4)

**Location**: `src/ts/layer4/Dereferencer.ts`

**Steps**:
1. Extract component/version from `scenario.ior`
2. Check if class already registered in TypeRegistry
3. If not, delegate to kernel's `componentLoad()` (check build, dynamic import, register)
4. Get class constructor
5. Create instance: `new ComponentClass()`
6. Initialize: `await instance.init(scenario)`
7. Return instance

**Pseudocode**:
```typescript
/**
 * Instantiate a UcpComponent from Scenario
 * 
 * Stage 2 of dereferencing: Scenario → Instance
 * Handles: class loading, build check, JsInterface registration
 * 
 * Uses Once.current to access kernel for componentLoad()
 */
async instantiate<T extends UcpComponent<M>, M extends Model>(
  scenario: Scenario<M>
): Promise<T> {
  // 1. Extract component info from scenario.ior
  const { component: componentName, version } = scenario.ior;
  
  // 2. Check TypeRegistry first (sync lookup)
  let ComponentClass = TypeRegistry.instance.classFromName(componentName);
  
  if (!ComponentClass) {
    // 3. Get kernel and delegate to componentLoad() (async)
    const kernel = Once.current;
    if (!kernel) {
      throw new Error('[Dereferencer] ONCE kernel not started');
    }
    ComponentClass = await kernel.componentLoad(componentName, version);
  }
  
  // 4. Create instance (sync: new)
  const instance = new ComponentClass() as T;
  
  // 5. Initialize with scenario (async: init)
  await instance.init(scenario as any);
  
  return instance;
}
```

### KD.4: Implement Dereferencer.dereference(reference) (1h)

**Purpose**: Universal dereferencing — handle all Reference<T> states (Layer 4)

**Location**: `src/ts/layer4/Dereferencer.ts`

**Pseudocode**:
```typescript
/**
 * Dereference a Reference<T> to its instance
 * 
 * Handles all three states:
 * - IOR string → load scenario → instantiate
 * - Scenario JSON → instantiate
 * - Instance → return as-is
 * 
 * Web4 P34: Reference<T> = IOR | Scenario | Instance | null
 */
async dereference<T extends UcpComponent<M>, M extends Model>(
  reference: Reference<T>
): Promise<T | null> {
  if (reference === null) {
    return null;
  }
  
  // Already an instance? (sync check)
  if (reference instanceof UcpComponent) {
    return reference as T;
  }
  
  // IOR string? (async: load + instantiate)
  if (typeof reference === 'string') {
    const scenario = await this.scenarioLoad<M>(reference);
    if (!scenario) return null;
    return await this.instantiate<T, M>(scenario);
  }
  
  // Scenario JSON? (async: instantiate only)
  if (typeof reference === 'object' && 'ior' in reference && 'model' in reference) {
    return await this.instantiate<T, M>(reference as Scenario<M>);
  }
  
  // Unknown format
  console.warn('[Dereferencer] Unknown reference format:', typeof reference);
  return null;
}
```
