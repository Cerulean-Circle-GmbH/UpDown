**Async Return Flow (per TRON guidance):**
```
Browser calls IOR → IORMethodRouter.route() [async, layer4]
    → Returns Scenario<T>
    → UcpController.scenarioReceive(scenario) [sync]
    → Object.assign(modelProxy, scenario.model) [sync]
    → Proxy set trap triggers viewsUpdateAll() [sync]
    → Views re-render [sync]
```

The async boundary is at layer4. When scenarios return, they trigger **sync** model updates which cause **sync** view re-rendering via UcpModel/UcpController pattern.
