# 🧱 Web4 Component Anatomy Checklist

**Purpose:** CHECK BEFORE RELEASE during component development.

- [ ] Component C1: Black Box - Internal implementation hidden, only layer3 interfaces exposed - [Details](./web4-component-anatomy-details.md#component-criteria) | [Example](../src/ts/layer3/ONCE.interface.ts)
- [ ] Component C2: Interfaces Exposed - Public API defined in `layer3/*.interface.ts` - [Details](./web4-component-anatomy-details.md#component-criteria) | [Example](../src/ts/layer3/)
- [ ] Component C3: Self-Contained - All code in versioned folder `/components/{Name}/{Version}/` - [Details](./web4-component-anatomy-details.md#component-criteria) | [Example](../)
- [ ] Component C4: Machine-Readable Descriptor - `{Name}.component.json` exists and valid - [Details](./web4-component-anatomy-details.md#component-descriptor) | [Generator](../src/ts/layer2/UnitDiscoveryService.ts)
- [ ] Descriptor D1: IS a Scenario - Has `{ ior, owner, model }` structure with UUIDv4 - [Details](./web4-component-anatomy-details.md#component-descriptor) | [Generator](../src/ts/layer2/UnitDiscoveryService.ts#L398-414)
- [ ] Descriptor D2: Lists Units - `model.units.css[]`, `model.units.templates[]` - [Details](./web4-component-anatomy-details.md#unit-categories) | [Generator](../src/ts/layer2/UnitDiscoveryService.ts#L294-305)
- [ ] Descriptor D3: Generator Created - NOT manually edited, generated by `UnitDiscoveryService.manifestUpdate()` - [Details](./web4-component-anatomy-details.md#generator-flow) | [Code](../src/ts/layer2/UnitDiscoveryService.ts#L390-424)
- [ ] Unit U1: Symlink Exists - `{file}.unit` symlink next to original file - [Details](./web4-component-anatomy-details.md#unit-structure) | [Example](../src/ts/layer5/views/css/FileItemView.css.unit)
- [ ] Unit U2: Points to Scenario - `→ scenarios/index/{uuid-path}/{uuid}.scenario.json` - [Details](./web4-component-anatomy-details.md#unit-structure) | [Generator](../src/ts/layer2/UnitDiscoveryService.ts#L346-373)
- [ ] Unit U3: Scenario Has UUIDv4 - `ior.uuid` is proper UUIDv4, NOT hallucinated strings - [Details](./web4-component-anatomy-details.md#unit-structure) | [Example](../../../../scenarios/index/f/c/2/d/b/fc2db7d0-c205-4472-ae4d-3c301ce95090.scenario.json)
- [ ] Unit U4: Model Has Origin - Git IOR: `ior:git:github.com/.../path/to/file` - [Details](./web4-component-anatomy-details.md#unit-structure) | [Generator](../src/ts/layer2/UnitDiscoveryService.ts)
- [ ] Unit U5: Model Has symlinkPaths - `symlinkPaths[]` for deduplication and discovery - [Details](./web4-component-anatomy-details.md#unit-structure)
- [ ] Lifecycle L1: Build - `src/sh/build.sh` compiles TypeScript - [Details](./web4-component-anatomy-details.md#lifecycle) | [Example](../src/sh/build.sh)
- [ ] Lifecycle L2: Install - `src/sh/install-deps.sh` handles dependencies - [Details](./web4-component-anatomy-details.md#lifecycle) | [Example](../src/sh/install-deps.sh)
- [ ] Lifecycle L3: Start - `npm start` or `./once` starts component - [Details](./web4-component-anatomy-details.md#lifecycle) | [Example](../src/sh/start.sh)
- [ ] Lifecycle L4: Test - `./once tootsie` runs all Tootsie tests - [Details](./web4-component-anatomy-details.md#lifecycle) | [Example](../test/tootsie/)
- [ ] Lifecycle L5: Clean - `src/sh/clean.sh` removes build artifacts - [Details](./web4-component-anatomy-details.md#lifecycle) | [Example](../src/sh/clean.sh)
