## Overview

**Core Vision**: IOR is the Universal Entry Point for ALL resources.

Every resource access (files, CSS, WebSocket, REST, scenarios) goes through IOR.
IOR internally resolves protocol chains and delegates to the appropriate Loader.

```
┌─────────────────────────────────────────────────────────────────┐
│                     IOR: Universal Entry Point                  │
├─────────────────────────────────────────────────────────────────┤
│  new IOR().init("ior:wss:https://host:port/ws").load()          │
│                           ↓                                     │
│  Protocol Chain: wss → https                                    │
│                           ↓                                     │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐             │
│  │ WebSocket   │  │  HTTPS      │  │  Scenario   │             │
│  │  Loader     │→ │  Loader     │→ │  Loader     │             │
│  └─────────────┘  └─────────────┘  └─────────────┘             │
└─────────────────────────────────────────────────────────────────┘
```

### **Phase I.1: Reference<T> → Web4 Class**

Current (duct tape type):
```typescript
export type Reference<T> = T | null;
```

Target (real Web4 class):
```typescript
export class Reference<T> {
    private _local: T | null = null;
    private _ior: IOR<T> | null = null;
    
    constructor() {}
    
    init(value: T | IOR<T> | null): this {
        if (value instanceof IOR) {
            this._ior = value;
        } else {
            this._local = value;
        }
        return this;
    }
    
    get isLocal(): boolean { return this._local !== null; }
    get isRemote(): boolean { return this._ior !== null; }
    get isNull(): boolean { return !this.isLocal && !this.isRemote; }
    
    async resolve(): Promise<T | null> {
        if (this.isLocal) return this._local;
        if (this.isRemote) return await this._ior!.load();
        return null;
    }
    
    get value(): T | null { return this._local; }
}
```

### **Phase I.6: Unit/File/Folder as Standard Scenarios**

**Vision**: Every file touched by StaticFileRoute becomes a Unit scenario

```
File Request: /ONCE/0.3.22.1/src/ts/layer5/views/css/ItemView.css
           ↓
StaticFileRoute:
  1. Serve file content
  2. Create Unit scenario (if not exists):
     {
       ior: { uuid: "...", class: "Unit", ... },
       model: {
         componentType: "CSSFile",
         componentIor: "ior:esm:/ONCE/0.3.22.1/css/ItemView",
         contentHash: "sha256:...",
         ...
       }
     }
  3. ServiceWorker caches Unit
           ↓
Next Request: ServiceWorker serves from cache
```
