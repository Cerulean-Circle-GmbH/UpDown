## P - PLAN

### Goal

Implement **Option D: Self-Replacing IOR Objects** from the research PDCA.

When a model property containing an IOR string is accessed:
1. UcpModel.get returns an **IOR instance** (not the string)
2. IOR triggers resolution in background (fire-and-forget)
3. When resolved, IOR **replaces itself** with the instance
4. Model change triggers view re-render automatically

### ISR.1: Add Parent-Aware Methods to IOR.ts (1h)

**File**: `src/ts/layer4/IOR.ts`

Add:
```typescript
/** Parent model for self-replacement */
private parentModel: Reference<object> = null;
private parentKey: Reference<string> = null;
private parentIndex: Reference<number> = null;

/**
 * Initialize with parent reference for self-replacement
 */
initWithParent(parent: object, key: string, index?: number): this {
  this.parentModel = parent;
  this.parentKey = key;
  this.parentIndex = index ?? null;
  return this;
}

/**
 * Resolve and replace self in parent model
 */
async resolveAndReplace<C>(): Promise<C> {
  const instance = await this.resolveInstance<C>();
  
  if (this.parentModel && this.parentKey) {
    const parent = this.parentModel as Record<string, unknown>;
    if (this.parentIndex !== null) {
      (parent[this.parentKey] as unknown[])[this.parentIndex] = instance;
    } else {
      parent[this.parentKey] = instance;
    }
  }
  return instance;
}
```

### ISR.2: Enhance UcpModel.get Handler (1h)

**File**: `src/ts/layer3/UcpModel.ts`

Modify `modelProxyCreate()`:
```typescript
get: function(target, property) {
  const value = (target as Record<string | symbol, unknown>)[property];
  
  // Not a string or not IOR? Return as-is
  if (typeof value !== 'string' || !value.startsWith('ior:')) {
    return value;
  }
  
  // IOR string → create self-replacing IOR
  const { IOR } = require('../layer4/IOR.js');
  const ior = new IOR().initRemote(value);
  ior.initWithParent(target, String(property));
  
  // Replace string with IOR in model
  (target as Record<string | symbol, unknown>)[property] = ior;
  
  // Fire-and-forget resolution
  ior.resolveAndReplace();
  
  return ior;
}
```
