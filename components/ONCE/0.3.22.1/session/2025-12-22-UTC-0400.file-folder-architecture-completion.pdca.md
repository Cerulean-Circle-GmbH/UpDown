```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          WEB4 FILE/FOLDER HIERARCHY                         │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  UcpComponent<Model>                                                        │
│       ↑                                                                     │
│  DefaultFile extends UcpComponent<FileModel>                                │
│       │  └── File is a LEAF (cannot have children)                          │
│       │  └── Has parent: Reference<Folder>                                  │
│       │  └── Content: Blob (IndexedDB native)                               │
│       │  └── contentHash: SHA-256 (for Artefact deduplication)              │
│       ↑                                                                     │
│  DefaultFolder extends UcpComponent<FolderModel>                            │
│       │      implements Tree<FileSystemNode>, Container<FileSystemNode>     │
│       │  └── Folder IS-A File AND CAN add children                          │
│       │  └── children: Collection<FileSystemNode>                           │
│       │  └── childAdd(file | folder): void                                  │
│       ↑                                                                     │
│  Component = Version Folder                                                 │
│       │  └── components/ONCE/0.3.22.1/ = Folder + UnitModel per file        │
│       │  └── scenarios/type/ONCE/0.3.22.1/ = Symlink mirror                 │
│       │  └── scenarios/index/{uuid[0-4]}/{uuid}.scenario.json = Source      │
│       ↑                                                                     │
│  Package = Component Container                                              │
│       └── Contains versioned components (Tree hierarchy)                    │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

export interface Folder extends File, Tree<FileSystemNode>, Container<FileSystemNode> {

## Emergent Properties (Small → Big Picture)

1. **File is a LEAF** → Cannot have children, has parent
2. **Folder IS-A File AND Tree** → Can add files/folders as children
3. **Component IS-A Version Folder** → Contains unique file units
4. **Package IS-A Component Container** → Contains versioned components
5. **Units link to scenarios/index/** → Content-addressable storage
6. **Type mirror in scenarios/type/** → Symlink-based organization
7. **PWA cache stores Units** → Offline capability emerges
8. **WebSocket syncs updates** → Real-time collaboration emerges
