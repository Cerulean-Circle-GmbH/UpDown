## **P - PLAN**

### **Objective**
Eliminate 25 scattered `fetch()` calls by centralizing all HTTP transport through `HTTPSLoader` (now in layer4). This ensures:
- **DRY**: Single implementation of fetch logic
- **P7 Compliance**: Async only in layer4
- **IOR Integration**: All requests can use IOR failover profiles

### **Web4 Component Architecture (L17)**
```
Layer 2 (Component) → owns model, manages views, triggers layer4
Layer 5 (Views) → render model, handle events, NEVER fetch
Layer 4 (Orchestrators) → ONLY async layer, fetch() lives HERE

Flow: View needs data → component.method() → layer4.async() → fetch → callback → model update → re-render
```

**Critical Rule**: Views NEVER do direct backend interaction. Only layer4 processes triggered by layer2 methods.

### **F.1: HTTPSLoader Singleton/Factory Pattern**

**Goal**: Make HTTPSLoader easily accessible without creating new instances everywhere.

**Option A: Singleton**
```typescript
// In HTTPSLoader.ts
private static instance: HTTPSLoader | null = null;

public static getInstance(): HTTPSLoader {
    if (!HTTPSLoader.instance) {
        HTTPSLoader.instance = new HTTPSLoader().init();
    }
    return HTTPSLoader.instance;
}

// Usage
const response = await HTTPSLoader.getInstance().load(url);
```

**Option B: Via ONCE Kernel**
```typescript
// Access via kernel (already available)
const loader = window.ONCE?.getLoader('https') as HTTPSLoader;
const response = await loader.load(url);
```

### **F.3: Layer5 View Fixes (ARCHITECTURAL VIOLATIONS)**

⚠️ **Views must NEVER fetch directly!** They must ask the component.

```typescript
// ❌ WRONG (current state)
class MyView extends DefaultView<MyComponent> {
    async render() {
        const response = await fetch('/api/endpoint');  // VIOLATION!
        const data = await response.json();
        this.renderData(data);
    }
}

// ✅ CORRECT (target state)
class MyView extends DefaultView<MyComponent> {
    render() {
        // View asks component for data
        this.component.loadEndpointData();  // Component handles async
    }
    
    onDataLoaded(data: any) {
        // Component calls this after layer4 resolves
        this.renderData(data);
    }
}

// Component (layer2)
class MyComponent extends UcpComponent {
    async loadEndpointData() {
        // Component triggers layer4
        const data = await this.orchestrator.fetchEndpoint();
        this.model.data = data;
        // Controller triggers re-render automatically via model change
    }
}
```
