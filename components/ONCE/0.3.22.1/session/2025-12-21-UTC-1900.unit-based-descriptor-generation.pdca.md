This is correct! The issue is:
1. Scenario is in `scenarios/ONCE/0.3.22.1/` (component type), NOT `scenarios/index/` (UUID-based)

const scenarioDir = path.join(scenariosRoot, componentName, componentVersion);

### DefaultStorage `saveScenario()` Pattern

From `Unit/0.3.0.5/src/ts/layer2/DefaultStorage.ts` (lines 68-94):

```typescript
async saveScenario(uuid: string, scenario: Scenario, symlinkPaths: string[]): Promise<void> {
  // Create UUID folder structure (5 levels deep)
  const folderPath = this.generateUUIDFolderPath(uuid);  // → scenarios/index/a/b/c/d/e/
  await fs.mkdir(folderPath, { recursive: true });

  // Create scenario file path
  const scenarioPath = join(folderPath, `${uuid}.scenario.json`);

  // Update scenario model with storage data
  const scenarioWithStorage = {
    ...scenario,
    model: {
      ...scenario.model,
      indexPath: scenarioPath,
      updatedAt: new Date().toISOString()
    }
  };

  // Save scenario to index
  await fs.writeFile(scenarioPath, JSON.stringify(scenarioWithStorage, null, 2), 'utf-8');

  // Create symbolic links
  for (const linkPath of symlinkPaths) {
    await this.createSymbolicLink(scenarioPath, linkPath);
  }
}
```

### DefaultUnit `toScenario()` Pattern

From `Unit/0.3.0.5/src/ts/layer2/DefaultUnit.ts` (lines 882-917):

```typescript
async toScenario(): Promise<Scenario<UnitModel>> {
  const componentVersion = await this.getComponentVersion();
  const componentName = await this.getComponentName();
  
  const scenario: Scenario = {
    ior: {
      uuid: this.model.uuid,
      component: componentName,
      version: componentVersion
    },
    owner: ownerData,
    model: this.model
  };

  // Create named link in current directory
  const links: string[] = [];
  if (this.model.name) {
    const linkFilename = `${this.convertNameToFilename(this.model.name)}.unit`;
    const namedLink = `${currentDir}/${linkFilename}`;
    links.push(namedLink);
  }
  
  await this.storage.saveScenario(this.model.uuid, scenario, links);
  await this.createAutomaticLinks();  // ontology and M3 type links
  
  return scenario;
}
```

### Required Architecture Changes

**Current State**:
```
components/ONCE/0.3.22.1/
├── ONCE.component.json → ../../../scenarios/ONCE/0.3.22.1/{uuid}.scenario.json  ✅
└── (no unit scenarios created)

scenarios/
├── ONCE/0.3.22.1/{uuid}.scenario.json  ← Component descriptor (NOT indexed)
└── index/  ← Empty (no unit scenarios)
```

**Target State**:
```
components/ONCE/0.3.22.1/
└── ONCE.component.json → ../../../scenarios/index/9/5/{uuid}.scenario.json

scenarios/
├── index/
│   ├── 9/5/{descriptor-uuid}.scenario.json  ← Component descriptor (INDEXED)
│   ├── a/b/{unit1-uuid}.scenario.json       ← Unit scenario for file 1
│   └── ...                                  ← Unit scenarios for all files
├── type/
│   └── ONCE/0.3.22.1/
│       ├── {descriptor-uuid}.scenario.json → ../../index/9/5/{descriptor-uuid}.scenario.json
│       ├── {unit1-uuid}.scenario.json → ../../index/a/b/{unit1-uuid}.scenario.json
│       └── ...  ← Symlinks to all units
└── ONCE/0.3.22.1/ ← (DEPRECATED: scenarios/type/ replaces this)
```

### Key Differences: Component vs Unit Storage

| Aspect | Component Descriptor | Unit |
|--------|---------------------|------|
| Primary Storage | `scenarios/{Component}/{Version}/` | `scenarios/index/{uuid[0]}/{uuid[1]}/` |
| Symlink in Component | `{Name}.component.json` | `{name}.unit` |
| Type Links | Not created | `scenarios/type/{Component}/{Version}/` |

**Required Changes**:
1. Move descriptor to `scenarios/index/` (UUID-based like units)
2. Create `unit` attribute: `Scenario<UnitModel>` for descriptor file itself
3. For each file in `model.units`, create `Scenario<UnitModel>` in `scenarios/index/`
4. Create symlinks in `scenarios/type/ONCE/0.3.22.1/` for all units

#### UD.4.1: Change Descriptor Storage to scenarios/index (UUID-based)

Current code stores to `scenarios/{Component}/{Version}/`:
```typescript
// WRONG: Current inline pattern
const scenarioDir = path.join(scenariosRoot, componentName, componentVersion);
```

Required code (follow DefaultStorage pattern):
```typescript
// CORRECT: Store in scenarios/index/{uuid[0]}/{uuid[1]}/
private generateUUIDFolderPath(uuid: string): string {
  const parts = uuid.split('-')[0]; // First segment of UUID
  const level1 = parts.substring(0, 1);
  const level2 = parts.substring(1, 2);
  return path.join(this.model.projectRoot, 'scenarios', 'index', level1, level2);
}

const scenarioDir = this.generateUUIDFolderPath(scenarioUuid);
```

#### UD.4.2: Create Type Symlinks

After saving to index, create symlink in `scenarios/type/`:

// Create type symlink: scenarios/type/{Component}/{Version}/{uuid}.scenario.json
const typeDir = path.join(scenariosRoot, 'type', componentName, componentVersion);
await fs.mkdir(typeDir, { recursive: true });

#### UD.4.3: Create Unit Scenarios for Each File

For each file discovered in `unitsDiscoverForDescriptor()`:
```typescript
// Create Scenario<UnitModel> for each file
async unitScenarioCreate(filePath: string, componentName: string, componentVersion: string): Promise<string> {
  const uuid = crypto.randomUUID();
  
  const unitModel: UnitModel = {
    uuid: uuid,
    name: path.basename(filePath),
    origin: `ior:local:file://${filePath}`,
    definition: '',
    typeM3: TypeM3.CLASS,
    indexPath: '', // Will be set by storage
    references: [{
      linkLocation: `ior:local:ln:file://${filePath}.unit`,
      linkTarget: `ior:local:file://scenarios/index/...`,
      syncStatus: SyncStatus.SYNCED
    }],
    createdAt: new Date().toISOString(),
    updatedAt: new Date().toISOString()
  };
  
  const scenario: Scenario<UnitModel> = {
    ior: { uuid, component: 'Unit', version: '0.3.0.5' },
    owner: ownerBase64,
    model: unitModel
  };
  
  // Save to scenarios/index/
  await this.saveToIndex(uuid, scenario);
  
  // Create type symlink
  await this.createTypeSymlink(uuid, componentName, componentVersion);
  
  return uuid;
}
```

#### UD.4.4: Add `unit` Attribute to Descriptor

const descriptorPath = path.join(componentRoot, `${componentName}.component.json`);

// Add unit attribute to the component scenario
