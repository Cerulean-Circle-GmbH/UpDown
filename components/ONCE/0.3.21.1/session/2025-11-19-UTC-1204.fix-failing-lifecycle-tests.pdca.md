### **Root Cause Identified**

**CRITICAL DISCOVERY:**
From test output analysis:
```bash
🔵 Started as CLIENT SERVER on port 8080
🔗 Connected to primary server at port 42777
```

**The actual problem:** Port 42777 is **already occupied** by a previous test run's primary server!

When tests start a "primary" server via `await primaryServer.startServer()`, the PortManager finds 42777 occupied and allocates 8080 instead, making it a **CLIENT** server, not PRIMARY.

**Result**:
- `getRegisteredServers()` returns [] because it's called on a CLIENT server (line 1114-1116 guards against this)
- Housekeeping never runs because only PRIMARY servers run housekeeping (line 118-125)
- All downstream tests fail because they expect a primary but got a client

**Step 2: Deeper Root Cause - stopServer() Hanging**
After freeing port 42777, tests started passing but still had 2 timeouts. Analysis revealed:

```typescript
// File: src/ts/layer2/ServerHierarchyManager.ts:1442-1458
// PROBLEM: httpServer.close() callback never fires if connections are active

if (this.httpServer) {
    return new Promise((resolve) => {
        this.httpServer!.close(async () => {  // ❌ Never completes with active connections
            console.log('🛑 Server stopped');
            resolve();
        });
    });
}
```

**Root Cause**: HTTP server won't close until ALL connections are terminated. Tests hang indefinitely waiting for `stopServer()` to complete.

---

**Step 3: Implemented Fix - Force Connection Closure + Timeout Fallback**
```typescript
// File: src/ts/layer2/ServerHierarchyManager.ts

async stopServer(): Promise<void> {
    // ... existing state updates ...

    // ✅ FIX 1: Close all WebSocket connections first
    if (this.wsServer) {
        this.wsServer.clients.forEach(client => {
            client.close();
        });
        this.wsServer.close();
    }

    if (this.httpServer) {
        return new Promise((resolve) => {
            // ✅ FIX 2: Force close all HTTP connections
            this.httpServer!.closeAllConnections?.();
            
            this.httpServer!.close(async () => {
                console.log('🛑 Server stopped');
                this.serverModel.state = LifecycleState.SHUTDOWN;
                await this.updateScenarioState(LifecycleState.SHUTDOWN);
                resolve();
            });
            
            // ✅ FIX 3: Timeout fallback (3s) in case close() never completes
            setTimeout(() => {
                console.warn('⚠️  Force closing HTTP server after timeout');
                this.serverModel.state = LifecycleState.SHUTDOWN;
                resolve();
            }, 3000);
        });
    }
}
```

**Changes**:
1. **Line 1439-1444**: Force close all WebSocket client connections before closing server
2. **Line 1449**: Call `closeAllConnections()` to force-close HTTP keep-alive connections
3. **Line 1467-1472**: Timeout fallback ensures Promise resolves even if close() hangs
