### **Web4 Layer Architecture**

**Layer 1 (Infrastructure)**: Hardware, networks, databases, OS
- ✅ Can be synchronous or async (infrastructure-specific)
- ✅ Examples: File I/O, network calls, database queries

**Layer 2 (Implementation)**: Business logic implementations
- ✅ MUST be synchronous
- ❌ NO async, NO await, NO promises
- ✅ Pure transformations and computations

**Layer 3 (Interface/Service Contracts)**: Pure interfaces
- ✅ MUST be synchronous
- ✅ No implementation, just contracts

**Layer 4 (Business Processes)**: Transaction chains with rollback
- ✅ ONLY layer that can use async
- ✅ Orchestrates Layer 1, 2, 3
- ✅ Handles all async I/O via Layer 1 services
- ✅ Injects data into Layer 2

**Layer 5 (User Experience)**: User interactions
- ✅ Can be async (event-driven)
- ✅ Calls Layer 4 processes

### **Target Architecture**

```
Layer 5 (CLI/Demo Hub) 
    ↓ (async call)
Layer 4 (ServerStartupProcess) ← ONLY ASYNC LAYER
    ↓ (inject owner data)
Layer 2 (DefaultONCE.toScenario) ← SYNCHRONOUS
    ↓ (use injected data)
Layer 1 (NodeOSInfrastructure) ← CAN BE ASYNC
```

### **Design Decisions**

**QA Decision D1: How to inject owner data into Layer 2?**
- **Option A**: Via constructor parameter (requires reconstruction)
- **Option B**: Via `init(scenario)` method (matches Web4 pattern) ✅

**Decision**: **D1b - Via init() method**
- Matches Web4 Empty Constructor + Scenario Init pattern
- Owner data becomes part of scenario model
- Clean separation: Layer 4 builds complete scenario, Layer 2 uses it

**QA Decision D2: Where to create Layer 4 process?**
- **Option A**: New file `layer4/ServerStartupProcess.ts` ✅

**Decision**: **D2a - New ServerStartupProcess.ts**
- Clear separation of concerns
- Follows Layer 4 naming convention (XxxProcess)
- Orchestrates server lifecycle with proper async handling

**QA Decision D3: How to handle User scenario generation?**
- **Option A**: Pre-generate in Layer 4, inject as string ✅
- **Option B**: Make DefaultUser.toScenario() synchronous
- **Option C**: Cache User scenarios globally

**Decision**: **D3a - Pre-generate in Layer 4**
- Keeps Layer 2 pure and synchronous

```typescript
// File: src/ts/layer4/ServerStartupProcess.ts (NEW)
import { DefaultONCE } from '../layer2/DefaultONCE.js';
import { DefaultUser } from '../layer2/DefaultUser.js';
import { NodeOSInfrastructure } from '../layer1/NodeOSInfrastructure.js';
import { ScenarioManager } from '../layer2/ScenarioManager.js';

/**
 * ServerStartupProcess - Layer 4 Business Process
 * ✅ ONLY layer that uses async
 * ✅ Orchestrates server startup with owner generation
 */
export class ServerStartupProcess {
  private once: DefaultONCE;
  private scenarioManager: ScenarioManager;
  
  constructor(once: DefaultONCE, scenarioManager: ScenarioManager) {
    this.once = once;
    this.scenarioManager = scenarioManager;
  }
  
  /**
   * Execute server startup process (async orchestration)
   */
  async execute(scenarioPath?: string): Promise<void> {
    // 1️⃣ Pre-generate owner data (async)
    const ownerData = await this.generateOwnerData();
    
    // 2️⃣ Inject owner data into Layer 2
    this.once.model.ownerData = ownerData;
    
    // 3️⃣ Start server (synchronous Layer 2 call)
    this.once.startServer(scenarioPath);
    
    // 4️⃣ Save scenario (Layer 2 now synchronous)
    const web4Scenario = this.once.toScenario();  // ✅ Synchronous!
    await this.saveScenarioViaInfrastructure(web4Scenario);
  }
  
  /**
   * Pre-generate owner data using User component
   */
  private async generateOwnerData(): Promise<string> {
    try {
      const infrastructure = new NodeOSInfrastructure();
      await infrastructure.init();
      const env = await infrastructure.detectEnvironment();
      const username = 'system';
      const projectRoot = this.scenarioManager.getProjectRoot();
      
      const user = await DefaultUser.create(username, infrastructure, projectRoot);
      const userScenario = await user.toScenario();
      const ownerJson = JSON.stringify(userScenario);
      return Buffer.from(ownerJson).toString('base64');
    } catch (error) {
      console.warn('⚠️  Failed to generate owner data, using fallback');
      // Return empty - Layer 2 will use its fallback
      return '';
    }
  }
  
  /**
   * Save scenario via Layer 1 infrastructure
   */
  private async saveScenarioViaInfrastructure(scenario: Scenario<LegacyONCEScenario>): Promise<void> {
    // Layer 4 calls Layer 1 for async I/O
    const fs = await import('fs');
    const path = await import('path');
    
    // Calculate path and save (Layer 1 I/O)
    const scenarioPath = await this.scenarioManager.calculateScenarioPath(scenario);
    const scenarioDir = path.dirname(scenarioPath);
    
    if (!fs.existsSync(scenarioDir)) {
      fs.mkdirSync(scenarioDir, { recursive: true });
    }
    
    fs.writeFileSync(scenarioPath, JSON.stringify(scenario, null, 2));
    console.log(`💾 Scenario saved: ${path.basename(scenarioPath)}`);
  }
}
```
