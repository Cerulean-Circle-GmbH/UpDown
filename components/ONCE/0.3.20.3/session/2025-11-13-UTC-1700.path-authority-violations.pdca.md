# PDCA: Path Authority Violations - Systematic Fix

**Date**: 2025-11-13 17:00 UTC  
**Component**: ONCE v0.3.20.3  
**Type**: Architecture Review & Systematic Fix  
**Status**: In Progress

## Problem Statement

During feature development (browser logging, CLI completion), we violated **Path Authority** principles multiple times by:
- Using `process.cwd()` instead of deriving paths from component location
- Hardcoding version numbers in paths and code
- Using fragile relative path calculations without backward links
- Relying on external state (environment variables, working directory) for paths

**Impact**: Code breaks when run from different directories, versions become hardcoded, and the system becomes fragile and non-portable.

## Root Cause

Lack of systematic application of TRUE Radical OOP Path Authority principles:
1. **The component instance that loads/creates another component is responsible for providing ALL paths to it**
2. **Never rely on process.cwd() or environment variables**
3. **Use backward links to query parent/creator for paths**
4. **Derive paths from import.meta.url, never guess**
5. **Pass paths down via init({ model: { ... }})**

### Step 2: Design Systematic Fix Pattern

**Pattern A: Component Root Calculation**
```typescript
// ❌ WRONG
const componentDir = process.cwd();

// ✅ CORRECT - Derive from own location
const dirname = path.dirname(fileURLToPath(import.meta.url));
const componentRoot = path.resolve(dirname, '../../../'); // dist/ts/layer2 -> root
```

**Pattern B: Version Extraction**
```typescript
// ❌ WRONG
version: '0.3.20.3'

// ✅ CORRECT - Extract from path
private get version(): string {
    const dirname = path.dirname(fileURLToPath(import.meta.url));
    const match = dirname.match(/ONCE\/(\d+\.\d+\.\d+\.\d+)/);
    return match ? match[1] : 'unknown';
}
```

**Pattern C: Backward Links**
```typescript
// ❌ WRONG - Manager calculates paths independently
class ScenarioManager {
    private get projectRoot(): string {
        return process.env.PROJECT_ROOT || path.resolve(...);
    }
}

// ✅ CORRECT - Manager asks component via backward link
class ScenarioManager {
    component?: DefaultONCE; // Backward link
    
    private get projectRoot(): string {
        if (this.component?.model.projectRoot) {
            return this.component.model.projectRoot;
        }
        // Fallback only if backward link not set
        const dirname = path.dirname(fileURLToPath(import.meta.url));
        return path.resolve(dirname, '../../../../../../');
    }
}
```

## Pattern: How to Apply Path Authority

### For Managers/Services
```typescript
class SomeManager {
    component?: ParentComponent; // ✅ Backward link
    
    private get version(): string {
        // 1. Try backward link first
        if (this.component?.model?.version) {
            return this.component.model.version;
        }
        
        // 2. Fallback: extract from own path
        const dirname = path.dirname(fileURLToPath(import.meta.url));
        const match = dirname.match(/ComponentName\/(\d+\.\d+\.\d+\.\d+)/);
        return match ? match[1] : 'unknown';
    }
    
    private get projectRoot(): string {
        // 1. Try backward link first
        if (this.component?.model?.projectRoot) {
            return this.component.model.projectRoot;
        }
        
        // 2. Fallback: calculate from own location
        const currentDir = dirname(fileURLToPath(import.meta.url));
        const componentRoot = resolve(currentDir, '../../../');
        const componentName = resolve(componentRoot, '..');
        const componentsDir = resolve(componentName, '..');
        return resolve(componentsDir, '..');
    }
}
```

### For Components
```typescript
class DefaultComponent {
    constructor() {
        // Extract version from directory path (single source of truth)
        const currentFileUrl = new URL(import.meta.url);
        const currentVersionDir = path.resolve(path.dirname(currentFileUrl.pathname), '..', '..', '..');
        const componentDirName = path.basename(currentVersionDir);
        const isVersionDir = /^\d+\.\d+\.\d+\.\d+$/.test(componentDirName);
        this.model.version = isVersionDir ? componentDirName : '0.0.0.0';
        
        // Set backward links
        this.someManager.component = this;
    }
}
```

### Anti-Patterns to Avoid
```typescript
// ❌ NEVER
const path = process.cwd();
const root = process.env.PROJECT_ROOT;
const version = '0.3.20.3'; // hardcoded
const path = '../../../../../../../'; // counting levels

// ✅ ALWAYS
const path = this.component.model.projectRoot; // backward link
const version = this.version; // dynamic property
const componentRoot = resolve(dirname(fileURLToPath(import.meta.url)), '../../../');
```
