🎭 ONCE v0.3.20.3 Demo Hub

Multi-Server Interactive Message Exchange Demo

All servers must be running before opening clients!

📋 Quick Start Instructions:

  1. Run demo: ./once demo 3 (or any number of clients)
  2. Wait for all servers to start (you'll see "Demo complete!")
  3. Click "Open All Clients" button below
  4. Use the buttons in each client to send real inter-server messages
  5. Watch messages flow between servers in real-time!
  6. Press q in terminal to stop all servers

📡 Active Servers:

🔄 Message Patterns Available in Each Client:

  • Broadcast: Primary server sends message to ALL clients
  • Relay: Client sends message to another client VIA primary server
  • P2P Direct: Client sends message DIRECTLY to another client (no relay)
  • ACK: Automatic acknowledgment on message receipt

💡 All buttons send REAL messages over WebSocket connections!
