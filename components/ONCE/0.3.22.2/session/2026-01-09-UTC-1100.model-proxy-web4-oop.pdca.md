## P — PLAN

### Problem Statement

Current `UcpModel` in `layer3/UcpModel.ts` uses **functional JavaScript Proxy handlers**:

```201:246:components/ONCE/0.3.22.2/src/ts/layer3/UcpModel.ts
  private modelProxyCreate(model: T): T {
    const ucpModel = this;
    
    return new Proxy(model, {
      set: function proxySet(target, property, value) {  // ← FUNCTIONAL
        const key = property as keyof T;
        (target as Record<keyof T, unknown>)[key] = value;
        // ... triggers viewUpdateCallback, persistenceCallback
        return true;
      },
      get: function proxyGet(target, property) {  // ← FUNCTIONAL
        const value = (target as Record<string | symbol, unknown>)[property];
        // ... ISR handling for IOR strings
        return value;
      }
    });
  }
```

**Root cause of `(this.model as any)` casts**: TypeScript loses type information through functional proxy handlers.

### Current Functional Handler Behaviors

| Handler | Behavior |
|---------|----------|
| `modelProxy.set` | 1. Sets value on target<br>2. Clears from updateObject<br>3. Triggers viewUpdateCallback()<br>4. Triggers persistenceCallback() |
| `modelProxy.get` | 1. Returns target[prop]<br>2. ISR: IOR string → IOR object<br>3. Array → arrayProxy wrap |
| `valueProxy.set` | 1. Sets value on target<br>2. Tracks in updateObjectInternal<br>3. NO callbacks |
| `valueProxy.get` | Simple return target[prop] |

## Web4 Radical OOP Solution

### Key Insight (from TRON)

`this.model` is ALWAYS typed as `TModel extends Model` where `TModel` is a strict interface:
- `ONCEPeerModel` for NodeJsOnce/BrowserOnce
- `Web4TSComponentModel` for DefaultWeb4TSComponent
- `CLIModel` for DefaultCLI
- etc.

### Proposed: Property Class

Migrate functional handlers to OOP `Property<T>` class with same behaviors:

```typescript
/**
 * Property<T> - Web4 Radical OOP property wrapper
 * Replaces functional Proxy handlers with typed OOP methods
 * 
 * @layer3
 */
export class Property<T> {
  private valueField: T;
  private parentModel: UcpModel<any>;
  private propertyKey: string;
  
  constructor() {
    // Empty - Web4 P6
  }
  
  /**
   * Initialize property
   * @param parent The UcpModel that owns this property
   * @param key Property name in the model
   * @param initialValue Initial value
   */
  init(parent: UcpModel<any>, key: string, initialValue: T): this {
    this.parentModel = parent;
    this.propertyKey = key;
    this.valueField = initialValue;
    return this;
  }
  
  /**
   * Get value (with ISR handling for IOR strings)
   */
  get value(): T {
    // ISR: If value is IOR string, replace with IOR object
    if (typeof this.valueField === 'string' && 
        (this.valueField as string).startsWith('ior:')) {
      const ior = new IOR().initRemote(this.valueField as string);
      ior.initWithParent(this.parentModel.rawModel, this.propertyKey);
      this.valueField = ior as unknown as T;
      ior.resolveAndReplace();
    }
    return this.valueField;
  }
  
  /**
   * Set value (triggers immediate updates)
   */
  set value(newValue: T) {
    this.valueField = newValue;
    this.parentModel.clearPendingChange(this.propertyKey);
    this.parentModel.triggerUpdates();
  }
  
  /**
   * Set value silently (tracked, no immediate update)
   */
  valueSetSilent(newValue: T): void {
    this.valueField = newValue;
    this.parentModel.trackChange(this.propertyKey, newValue);
  }
}
```

### Proposed: UcpModel Refactor

```typescript
/**
 * UcpModel<TModel> - Web4 Radical OOP Model Wrapper
 * 
 * TModel must be a strict model interface (ONCEPeerModel, Web4TSComponentModel, etc.)
 * Properties are typed Property<T> instances, not functional handlers.
 */
export class UcpModel<TModel extends Model> {
  
  /** Raw model data */
  private rawModelField!: TModel;
  
  /** Typed properties map */
  private propertiesField!: Map<keyof TModel, Property<TModel[keyof TModel]>>;
  
  /** View update callback */
  private viewUpdateCallbackField!: () => void;
  
  /** Persistence callback */
  private persistenceCallbackField: Reference<() => void> = null;
  
  /** Change tracking */
  private updateObjectField: Partial<TModel> = {};
  
  constructor() {
    // Empty - Web4 P6
  }
  
  /**
   * Initialize with typed model
   * Creates Property<T> instances for each model property
   */
  init(model: TModel, onViewUpdate: () => void): this {
    this.rawModelField = model;
    this.viewUpdateCallbackField = onViewUpdate;
    this.propertiesField = new Map();
    
    // Create typed Property for each model key
    for (const key of Object.keys(model) as (keyof TModel)[]) {
      const prop = new Property<TModel[typeof key]>()
        .init(this, key as string, model[key]);
      this.propertiesField.set(key, prop);
    }
    
    return this;
  }
  
  /**
   * Get typed property accessor
   * Returns Property<T> where T is the type of TModel[K]
   */
  property<K extends keyof TModel>(key: K): Property<TModel[K]> {
    let prop = this.propertiesField.get(key);
    if (!prop) {
      // Create on-demand for properties not in initial model
      prop = new Property<TModel[K]>()
        .init(this, key as string, undefined as TModel[K]);
      this.propertiesField.set(key, prop);
    }
    return prop as Property<TModel[K]>;
  }
  
  /**
   * Get model value directly (typed)
   * Same as property(key).value but more concise
   */
  get<K extends keyof TModel>(key: K): TModel[K] {
    return this.property(key).value;
  }
  
  /**
   * Set model value (triggers updates)
   */
  set<K extends keyof TModel>(key: K, value: TModel[K]): void {
    this.property(key).value = value;
  }
  
  // ... rest of UcpModel methods (commit, rollback, merge, etc.)
}
```

### Usage Pattern Change

```typescript
// BEFORE (loses type):
this.model.componentRoot  // TypeScript: any

// AFTER Option A (explicit property access):

// AFTER Option B (typed get/set):
this.ucpModel.get('componentRoot')  // TypeScript: string

// AFTER Option C (backwards compatible Proxy wrapper):
```
