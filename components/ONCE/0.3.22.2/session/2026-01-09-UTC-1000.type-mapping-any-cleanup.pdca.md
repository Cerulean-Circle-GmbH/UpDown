## P — PLAN

### Root Cause (from TRON review)

The `(this.model as any)` casts are NOT caused by missing interface properties.

**The real problem**: UcpModel (ModelProxy) uses **functional JavaScript Proxy pattern** instead of Web4 Radical OOP. This forces TypeScript to lose type information.

**Solution**: Fix the Proxy implementation FIRST → then `as any` casts become unnecessary.

See: [ModelProxy — Web4 Radical OOP](./2026-01-09-UTC-1100.model-proxy-web4-oop.pdca.md)

**Total `as any)` casts found**: 136 across 10 files in layer2

### Issues in `web4-component-anatomy-details.md`

The documentation shows these incorrect accessor patterns:

```typescript
// WRONG: Uses string and any cast
get componentRoot(): string {
  return (this.model as any)?.componentRoot ?? '';
}

get targetComponentRoot(): string {
  return (this.model as any)?.targetComponentRoot ?? this.componentRoot;
}
```

### Questions for TRON

**Q1: Path types — Should they be `Folder` (runtime `FolderJs`) instead of `string`?**

```typescript
// Current (string):
get componentRoot(): string { ... }
get targetComponentRoot(): string { ... }
get projectRoot(): string { ... }

// Proposed (Folder):
get componentRoot(): FolderJs { ... }
get targetComponentRoot(): FolderJs { ... }
get projectRoot(): FolderJs { ... }
```

If `Folder`, how to handle lazy initialization? Folder needs async creation.

**Q2: What about CLI model paths?**

`CLIModel.projectRoot` is currently `string`. Should it also be `FolderJs`?

## Solution (from TRON)

**NOT** interface extensions or Folder types. The root cause is the Proxy implementation.

### Correct Solution Path

1. **First**: Fix ModelProxy with Web4 Radical OOP pattern
   - See: [ModelProxy PDCA](./2026-01-09-UTC-1100.model-proxy-web4-oop.pdca.md)
   - `class Property<T>` with `get/set` instead of functional handlers
   - `class ModelProxy<TModel>` with typed property access

2. **Then**: `as any` casts become unnecessary because TypeScript can infer types

3. **For Unit**: Use existing Web4 component types (they exist!)

4. **For Artefact**: Create as Web4 component (future TODO)

### Path Types Decision

**KISS**: Keep paths as `string` — NOT Folder objects.
But remove the `any` cast by fixing the Proxy, not by changing the type.
