## **📋 PLAN**

### **Current State (Problem)**

```typescript
// UcpComponent.ts
abstract class UcpComponent<TModel> {
  protected abstract modelDefault(): TModel;  // ← UNNECESSARY ABSTRACTION
  
  scenarioDefault(): Scenario<TModel> {
    return {
      ior: { uuid, component, version },      // ← CORRECT (from UcpComponent)
      owner: this.owner,                       // ← CORRECT (from UcpComponent)
      model: this.modelDefault()               // ← CALLS CHILD'S modelDefault()
    };
  }
  
  init(scenario?: Scenario<TModel>): this {
    const s = scenario ?? this.scenarioDefault();
    // ... initialize
  }
}

// DefaultWeb4TSComponent.ts
class DefaultWeb4TSComponent extends UcpComponent<Web4TSComponentModel> {
  protected modelDefault(): Web4TSComponentModel {  // ← SEPARATE METHOD = BAD
    return { uuid, name, version, ... };
  }
  
  init(scenario?: Scenario<Web4TSComponentModel>): this {
    super.init(scenario);
    // ... additional setup
  }
}
```

**Problem:** 
- `modelDefault()` is a separate method — unnecessary indirection
- `scenarioDefault()` assembles ior/owner correctly but calls child's `modelDefault()`
- Child classes must implement TWO things: `modelDefault()` AND override `init()`

### **Target State (Solution)**

```typescript
// UcpComponent.ts
abstract class UcpComponent<TModel> {
  // NO modelDefault()!
  
  scenarioDefault(): Scenario<TModel> {
    return {
      ior: {
        uuid: this.uuidCreate(),
        component: this.constructor.name,
        version: this.componentVersion
      },
      owner: this.owner,
      model: null  // ← Not initialized, child assigns
    };
  }
  
  init(scenario?: Scenario<TModel>): this {
    // ... initialize with scenario
  }
}

// DefaultWeb4TSComponent.ts
class DefaultWeb4TSComponent extends UcpComponent<Web4TSComponentModel> {
  // NO modelDefault()!
  
  init(scenario?: Scenario<Web4TSComponentModel>): this {
    if (!scenario) {
      scenario = super.scenarioDefault();  // ← Gets correct ior/owner
      const version = new SemanticVersion().init();
      scenario.model = {                    // ← Plain JSON assignment
        uuid: randomUUID(),
        name: 'Web4TSComponent',
        component: 'Web4TSComponent',
        version: version,
        componentRoot: '',
        targetDirectory: '',
        isTestIsolation: false,
        displayName: 'Web4TSComponent',
        displayVersion: version.toString(),
        isDelegation: false,
      };
    }
    
    return super.init(scenario);
  }
}
```

### **Benefits**
1. **KISS** — model is null until assigned, then plain JSON
2. **No indirection** — no separate `modelDefault()` method
3. **Correct ior/owner** — comes from `scenarioDefault()` in parent
4. **DRY** — one place, one assignment

## **MDE.P2: toScenario() Persistence (IndexedDB + SW Cache)**

### **Current Architecture**

```
UcpModel:
├── persistenceCallback: Reference<() => void> = null
├── persistenceCallbackSet(callback) - registers callback
└── On model change → calls persistenceCallback()

BrowserScenarioStorage (PersistenceManager):
├── scenarioSave(uuid, scenario, symlinkPaths) - saves to IndexedDB
└── Uses RelatedObjects pattern for lookup

UcpComponent.toScenario():
└── Returns { ior, owner, model, unit } - current state
```

### **Required Behavior**

1. **toScenario() returns EXACT current state** — already works (model is proxy)
2. **Auto-persist to IndexedDB** — when model changes, save toScenario()
3. **Auto-persist to SW Cache** — for PWA offline support

### **Solution**

Wire persistence in UcpComponent.init():

```typescript
// UcpComponent.init()
init(scenario?: Scenario<TModel>): this {
  // ... existing init logic ...
  
  // Wire persistence callback to UcpModel
  this.ucpModel.persistenceCallbackSet(this.scenarioPersist.bind(this));
  
  return this;
}

/**
 * Persist current scenario to IndexedDB + SW Cache
 * Called automatically when model changes (via UcpModel proxy)
 */
protected scenarioPersist(): void {
  const scenario = this.toScenario();  // Current state
  const uuid = scenario.ior.uuid;
  
  // Get PersistenceManager from RelatedObjects (if available)
  const pm = this.controller.relatedObjectLookupFirst(PersistenceManager);
  if (pm) {
    // Fire-and-forget async save
    pm.scenarioSave(uuid, scenario, this.symlinkPathsBuild());
  }
  
  // SW Cache: Post to ServiceWorker for caching
  if (typeof navigator !== 'undefined' && navigator.serviceWorker?.controller) {
    navigator.serviceWorker.controller.postMessage({
      type: 'CACHE_SCENARIO',
      uuid,
      scenario
    });
  }
}
```

### **Invariant**
- `toScenario()` ALWAYS returns exact current model state
- Model changes → auto-save to IndexedDB + SW Cache
- No manual save required
