## ✅ COMPLETED: CPA.3 — Remove `as any` Backward Compat Fallbacks

**Date Completed:** 2026-01-09  
**Root Cause:** UcpComponent accessors had backward compat fallbacks that required `as any`

### Problem (FIXED)

```typescript
// UcpComponent.ts lines 272-276
get projectRoot(): string {
  return this.cliField?.model?.projectRoot 
    ?? (this.model as any)?.projectRoot   // ← BACKWARD COMPAT — needs as any
    ?? '';
}
```

The `(this.model as any)` is needed because:
- `UcpComponent<TModel extends Model>` is generic
- `Model` base interface doesn't have `projectRoot`
- This fallback was for when CLI is not set

**CLI IS the path authority — remove the fallbacks!**

## 🎯 Architecture (Reference)

```
┌─────────────────────────────────────────────────────────────────┐
│                      UcpComponent (Base Class)                   │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │ private cliField: Reference<DefaultCLI>                    │ │
│  │ get cli() / set cli()                                      │ │
│  │ get projectRoot() → this.cli.model.projectRoot             │ │
│  └───────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
          ▲                        ▲
          │ extends                │ extends
┌─────────┴─────────┐  ┌──────────┴──────────┐
│  DefaultWeb4TS    │  │   NodeJsOnce        │
│  Component        │  │                     │
├───────────────────┤  ├─────────────────────┤
│ override          │  │ override            │
│ componentRoot()   │  │ componentRoot()     │
└───────────────────┘  └─────────────────────┘
```

**Key Principles:**
1. **CLI is Path Authority** — `this.projectRoot` → `this.cli.model.projectRoot`
2. **No backward compat fallbacks** — CLI must be set
3. **Subclass overrides** — `componentRoot` in subclasses where model is known
