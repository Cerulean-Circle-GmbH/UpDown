## Problem Statement

### Discovery

When changing `Folder.children` from `File[]` to `Collection<File>`:

```typescript
// Caused 12 compilation errors:
src/ts/layer2/DefaultFolder.ts(315,17): error TS2339: Property 'model' does not exist on type 'string | FileJs | IORType<FileJs>'.
```

**Root Cause**: Existing code assumes `children` contains only resolved instances.

### Current Workaround (DRY Violation)

```typescript
interface Folder extends File {
  readonly children: File[];           // Resolved only (filtered)
  readonly childReferences: Collection<File>;  // Raw (may have IORs)
}
```

**Target State (DRY)**:

```typescript
interface Folder extends File {
  readonly children: Collection<File>;  // ISR handles resolution
}
```

---

## User Feedback (Quoted)

> "as all references by default are level one preloaded, the ucpComponent or the UcpModel should track if all references have been resolved or its during preloading. so methods that rely on everything preloaded can await complete preloading in layer4 and then continue with synchronous calls to other layers."

## Analysis

### Layer Responsibilities

| Layer | Async? | Responsibility |
|-------|--------|----------------|
| **Layer 4** (Orchestrator) | ✅ async | Await `model.awaitResolved()` |
| **Layer 2** (Component) | ❌ sync | Assume references are instances |
| **Layer 3** (Model) | ❌ sync | Track resolution state |
| **Layer 5** (View) | ❌ sync | Render resolved model |

### UcpModel Resolution State

```typescript
interface ResolutionState {
  /** True if all LazyReference properties are resolved to instances */
  isResolved: boolean;
  
  /** True if resolution is in progress */
  isResolving: boolean;
  
  /** Number of unresolved references */
  unresolvedCount: number;
  
  /** Promise that resolves when all references are resolved */
  awaitResolved(): Promise<void>;
}
```

## Plan

### RT.1: Add resolution tracking to UcpModel (2h)

**File**: `src/ts/layer3/UcpModel.ts`

```typescript
// ADD: Resolution state tracking
private unresolvedCount: number = 0;
private pendingResolutions: number = 0;
private resolveCallback: Reference<() => void> = null;

get isResolved(): boolean {
  return this.unresolvedCount === 0;
}

get isResolving(): boolean {
  return this.pendingResolutions > 0;
}

awaitResolved(): Promise<void> {
  if (this.isResolved) return Promise.resolve();
  return new Promise(resolve => {
    this.resolveCallback = resolve;
  });
}

// Called when a reference resolves
private onReferenceResolved(): void {
  this.unresolvedCount--;
  this.pendingResolutions--;
  if (this.unresolvedCount === 0 && this.resolveCallback) {
    this.resolveCallback();
    this.resolveCallback = null;
  }
}
```

### RT.2: Count unresolved references on model init (1h)

**File**: `src/ts/layer3/UcpModel.ts`

```typescript
// In init() or constructor
private countUnresolvedReferences(model: T): number {
  let count = 0;
  for (const key of Object.keys(model)) {
    const value = model[key as keyof T];
    if (typeof value === 'string' && value.startsWith('ior:')) {
      count++;
    }
    if (Array.isArray(value)) {
      for (const item of value) {
        if (typeof item === 'string' && item.startsWith('ior:')) {
          count++;
        }
      }
    }
  }
  return count;
}
```

get(target, prop) {
  const value = target[prop as keyof T];
  
  if (typeof value === 'string' && value.startsWith('ior:')) {
    // Start resolution
    this.pendingResolutions++;
    IOR.resolveAndReplace(target, prop, value)

**Pattern**: Sync methods guard, log, return this (no exceptions)

```typescript
// layer2/UcpComponent.ts
protected guardReady(): boolean {
  if (!this.model.isResolved) {
    console.warn(
      `[${this.constructor.name}] Method called before resolution. ` +
      `Skipping execution.`
    );
    return false;
  }
  return true;
}

get isReady(): boolean {
  return this.model.isResolved;
}
```

**Usage in methods**:

```typescript
// layer2/DefaultFolder.ts
open(): this {
  if (!this.guardReady()) return this;  // Skip if not ready
  this.model.isOpen = true;
  return this;
}
```
