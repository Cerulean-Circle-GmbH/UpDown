## Root Cause Analysis

### The Actual Error

```
Failed to create unit from file: Cannot set properties of null (setting 'uuid')
```

This occurred because:

1. `DefaultUnit.from()` calls `this.folderUnitCreate()`
2. `folderUnitCreate()` does `this.model.uuid = this.uuidCreate()`
3. But `this.model` returns `null` if `ucpModel` is not initialized

### Why `initSync({})` Didn't Work

Looking at `UcpComponent.initSync()`:
```typescript
initSync(scenario?: { model?: TModel }): this {
  if (this.ucpModel !== null) {
    // Already initialized — skip
    return this;
  }
  const rawModel = scenario?.model ?? this.modelDefault();
  // ... creates ucpModel
}
```

The issue is that `initSync({})` passes an empty object `{}`, but:
- `scenario?.model` is `undefined` (not null)
- So it correctly calls `this.modelDefault()`
- But `modelDefault()` in `DefaultUnit` calls `this.uuidCreate()`
- And `uuidCreate()` works fine (has fallback to static provider)

**The REAL issue:** I was using dynamic import:
```typescript
const { DefaultUnit } = await import('./DefaultUnit.js');
const unit = new DefaultUnit();
unit.initSync({});
```

The dynamic import might be causing the class to not have proper prototype chain or static initialization.

## How I Should Have Done It

### Correct Pattern: Static Import + Proper Init

```typescript
// At top of file
import { DefaultUnit } from './DefaultUnit.js';

// In foldersDiscover()
private async foldersDiscover(componentRoot: string, baseDir: string): Promise<void> {
  const fullBase = path.join(componentRoot, baseDir);
  if (!fs.existsSync(fullBase)) return;
  
  const excludeDirs = ['node_modules', '.git', 'coverage', 'test-results'];
  
  const discoverDir = async (dir: string, relativePath: string) => {
    const folderUnitPath = path.join(dir, '°folder.unit');
    if (!fs.existsSync(folderUnitPath)) {
      // Use DefaultUnit.from() — P8: DRY
      const unit = new DefaultUnit();
      unit.initSync({});
      await unit.from(dir);
    }
    
    const entries = await fs.promises.readdir(dir, { withFileTypes: true });
    for (const entry of entries) {
      if (!entry.isDirectory()) continue;
      if (excludeDirs.includes(entry.name)) continue;
      await discoverDir(path.join(dir, entry.name), path.join(relativePath, entry.name));
    }
  };
  
  await discoverDir(fullBase, baseDir);
}
```
