### **Key Design Decisions**

**QA Decision D1: How to structure environment detection?**
- **Option A**: Static method `ONCE.create()` detects and returns correct implementation ✅
- **Option B**: Constructor detects and delegates internally
- **Option C**: Separate factory class

**Decision**: **D1a - Static factory method**
- Clean API: `const once = await ONCE.create();`
- Returns correct subclass instance
- Maintains backward compatibility with `new DefaultONCE()`
- Follows Web4 pattern: `DefaultUser.create()`

**QA Decision D2: Router architecture?**
- **Option A**: Express.js-style router with middleware ✅
- **Option B**: Simple path-to-handler map
- **Option C**: Regex-based routing

**Decision**: **D2a - Express-style with middleware**
- Dynamic route registration
- Middleware support for auth, logging, etc.
- Clean separation of concerns
- Familiar pattern for developers

**QA Decision D3: EAMD.ucp creation timing?**
- **Option A**: On primary server start (as specified) ✅
- **Option B**: On first HTTP request
- **Option C**: During component installation

**Decision**: **D3a - Primary server start**
- User requirement explicit
- Primary server is authoritative
- Client servers don't need to manage www-root

**QA Decision D4: Browser script migration strategy?**
- **Option A**: Keep functional scripts, add OOP classes alongside (gradual) ✅
- **Option B**: Replace all scripts with classes immediately
- **Option C**: Hybrid: classes call functional scripts

**Decision**: **D4a - Gradual migration**
- Zero breaking changes
- Old functional scripts remain for backward compatibility
- New OOP classes introduced alongside
- Demo pages updated to use OOP
- Tests verify both work

**QA Decision D5: How to handle route conflicts?**
- **Option A**: Last registered route wins
- **Option B**: First registered route wins ✅
- **Option C**: Throw error on conflict

**Decision**: **D5b - First wins (with warning)**
- Predictable behavior
- Log warning if conflict detected
- Allows override by registering first

   export class HttpRouter {
     private routes: Route[] = [];
     
     constructor() {}
     
     public addRoute(route: Route): this {
       // Check for conflicts
       const existing = this.routes.find(r => 
         r.getPath() === route.getPath() && 
         r.getMethod() === route.getMethod()
       );
       
       if (existing) {
         console.warn(`⚠️ Route conflict: ${route.getMethod()} ${route.getPath()}`);
         return this; // First wins
       }
       
       this.routes.push(route);
       return this;
     }
     
     public route(req: IncomingMessage): RouteResult {
       const url = new URL(req.url || '/', `http://${req.headers.host}`);
       const method = req.method || 'GET';
       
       for (const route of this.routes) {
         const match = route.match(url.pathname, method);
         if (match.matched) {
           return { route, match, url };
         }
       }
       
       return { route: null, match: { matched: false }, url };
     }
     
     public getRoutes(): Route[] {
       return [...this.routes];
     }
   }

2. **`src/ts/layer1/HttpRoute.ts`** (NEW)
   ```typescript
   export type HttpMethod = 'GET' | 'POST' | 'PUT' | 'DELETE' | 'PATCH' | 'OPTIONS' | 'HEAD';
   export type RouteHandler = (req: any, res: any, params: RouteParams) => Promise<void> | void;
   
   export interface RouteParams {
     [key: string]: string;
   }
   
   export interface RouteMatch {
     matched: boolean;
     params?: RouteParams;
   }
   
   export class HttpRoute {
     private path: string;
     private method: HttpMethod;
     private handler: RouteHandler;
     private pathRegex: RegExp;
     private paramNames: string[] = [];
     
     constructor(path: string, method: HttpMethod, handler: RouteHandler) {
       this.path = path;
       this.method = method;
       this.handler = handler;
       this.pathRegex = this.compilePathToRegex(path);
     }
     
     private compilePathToRegex(path: string): RegExp {
       // Convert /users/:id/posts/:postId to regex
       // Extract param names like :id, :postId
       const paramPattern = /:([a-zA-Z_][a-zA-Z0-9_]*)/g;
       let match;
       
       while ((match = paramPattern.exec(path)) !== null) {
         this.paramNames.push(match[1]);
       }
       
       // Replace :param with capture group
       const regexPattern = path
         .replace(/\//g, '\\/')
         .replace(/:([a-zA-Z_][a-zA-Z0-9_]*)/g, '([^/]+)');
       
       return new RegExp(`^${regexPattern}$`);
     }
     
     public match(pathname: string, method: string): RouteMatch {
       if (method !== this.method) {
         return { matched: false };
       }
       
       const match = this.pathRegex.exec(pathname);
       if (!match) {
         return { matched: false };
       }
       
       // Extract params
       const params: RouteParams = {};
       for (let i = 0; i < this.paramNames.length; i++) {
         params[this.paramNames[i]] = match[i + 1];
       }
       
       return { matched: true, params };
     }
     
     public async handle(req: any, res: any, params: RouteParams): Promise<void> {
       await this.handler(req, res, params);
     }
     
     public getPath(): string {
       return this.path;
     }
     
     public getMethod(): HttpMethod {
       return this.method;
     }
   }
   ```

   /**
    * EAMD.ucp Manager - Creates and manages unified www-root directory
    * EAMD = Enterprise Architecture Management Directory (Web4 standard)
    */
   export class EAMDManager {
     private projectRoot: string;
     private eamdPath: string;
     
     constructor(projectRoot: string) {
       this.projectRoot = projectRoot;
       this.eamdPath = path.join(projectRoot, 'EAMD.ucp');
     }
     
     /**
      * Create EAMD.ucp directory structure with symlinks
      * Called on primary server start
      */
     public async ensureEAMDStructure(): Promise<void> {
       // Create EAMD.ucp if not exists
       if (!fs.existsSync(this.eamdPath)) {
         fs.mkdirSync(this.eamdPath, { recursive: true });
         console.log(`📁 Created EAMD.ucp directory: ${this.eamdPath}`);
       }
       
       // Create symlink to components
       const componentsLink = path.join(this.eamdPath, 'components');
       const componentsTarget = path.join(this.projectRoot, 'components');
       await this.ensureSymlink(componentsLink, componentsTarget, '../components');
       
       // Create symlink to scenarios
       const scenariosLink = path.join(this.eamdPath, 'scenarios');
       const scenariosTarget = path.join(this.projectRoot, 'scenarios');
       await this.ensureSymlink(scenariosLink, scenariosTarget, '../scenarios');
       
       console.log(`✅ EAMD.ucp structure ready`);
       console.log(`   📂 ${this.eamdPath}/components → ../components`);
       console.log(`   📂 ${this.eamdPath}/scenarios → ../scenarios`);
     }
     
     private async ensureSymlink(linkPath: string, targetPath: string, relativeTarget: string): Promise<void> {
       if (fs.existsSync(linkPath)) {
         const stats = fs.lstatSync(linkPath);
         if (stats.isSymbolicLink()) {
           // Already exists and is symlink
           return;
         } else {
           // Exists but not symlink - error
           console.warn(`⚠️ ${linkPath} exists but is not a symlink!`);
           return;
         }
       }
       
       // Create symlink
       fs.symlinkSync(relativeTarget, linkPath);
       console.log(`🔗 Created symlink: ${path.basename(linkPath)} → ${relativeTarget}`);
     }
     
     public getEAMDPath(): string {
       return this.eamdPath;
     }
   }

   export class DefaultONCE implements ONCE {

     /**
      * Static factory method - creates environment-appropriate ONCE instance
      * ✅ Web4 Pattern: Environment detection → implementation selection
      */
     public static async create(scenario?: LegacyONCEScenario): Promise<ONCE> {
       // Detect environment
       const env = DefaultONCE.detectEnvironmentStatic();
       
       let once: ONCE;
       
       if (env.platform === 'browser') {
         // Dynamic import for browser
         const { ONCEBrowserClient } = await import('./ONCEBrowserClient.js');
         once = new ONCEBrowserClient();
       } else if (env.platform === 'node') {
         // Dynamic import for Node.js
         const { ONCENodeServer } = await import('./ONCENodeServer.js');
         once = new ONCENodeServer();
       } else {
         // Fallback to DefaultONCE
         once = new DefaultONCE();
       }
       
       // Initialize with scenario
       await once.init(scenario);
       
       // Environment-specific initialization
       if ('initEnvironment' in once) {
         await (once as any).initEnvironment();
       }
       
       return once;
     }
     
     private static detectEnvironmentStatic(): EnvironmentInfo {
       // Browser detection
       if (typeof window !== 'undefined' && typeof document !== 'undefined') {
         return {
           platform: 'browser',
           version: navigator.userAgent,
           capabilities: ['websocket', 'p2p'],
           isOnline: navigator.onLine
         };
       }
       
       // Node.js detection
       if (typeof process !== 'undefined' && process.versions?.node) {
         return {
           platform: 'node',
           version: process.version,
           capabilities: ['server', 'websocket', 'p2p'],
           isOnline: true
         };
       }
       
       // Worker detection
       if (typeof self !== 'undefined' && typeof importScripts === 'function') {
         return {
           platform: 'worker',
           version: 'unknown',
           capabilities: ['websocket'],
           isOnline: true
         };
       }
       
       throw new Error('Unknown environment');
     }
   }
