# P2P Multi-Server Plan Without Protocol Messages

**Parent PDCA**: 2025-11-21-UTC-1900.iteration-01.6-once-architecture-consolidation.pdca.md  
**Date**: 2025-11-21  
**Topic**: Detailed implementation plan for protocol-less P2P multi-server functionality

---

## **Challenge**

How to maintain P2P functionality (primary 42777 + client servers) using object replication instead of message protocols?

## **Solution**

Objects register themselves by replicating to the primary, not by sending registration messages!

---

## **Current Architecture (Message-Based - WRONG)**

```typescript
// ❌ Client server sends registration message
const registrationMessage = {
  type: 'server-register',
  from: { uuid: 'abc123', host: 'localhost', port: 8080 },
  data: { capabilities: [...], state: 'running' }
};
ws.send(JSON.stringify(registrationMessage));

// ❌ Primary handles message
ws.on('message', (data) => {
  const msg = JSON.parse(data);
  if (msg.type === 'server-register') {
    serverRegistry.set(msg.from.uuid, msg.data);
  }
});
```

**Problem**: Functional message handlers, protocol types, no OOP!

---

## **Web4 Architecture (Object Replication - RIGHT)**

**Key Insight**: Servers ARE objects! Just replicate them!

```typescript
// ✅ Client server replicates itself to primary
const myScenario = await this.toScenario();  // HTTPSServer hibernates itself
ws.send(JSON.stringify(myScenario));         // Send scenario (no protocol!)

// ✅ Primary receives and creates clone
ws.on('message', (data) => {
  const scenario = JSON.parse(data) as Scenario<HTTPSServerModel>;
  
  // Create replica of client server locally!
  const clientReplica = new HTTPSServer();
  clientReplica.init(scenario);  // SAME uuid, SAME state!
  
  // Store replica in registry
  this.serverRegistry.set(clientReplica.ior.uuid, clientReplica);
});
```

**Result**: Primary has **clones** of all client servers, not just metadata!

## **Architecture Components**

### **1. ServerRegistry (Replaces serverRegistry: Map<UUID, ServerEntry>)**

```typescript
// src/ts/layer2/ServerRegistry.ts (NEW)
/**
 * ServerRegistry - Stores clones of registered servers
 * Web4 Pattern: Registry contains actual object replicas, not metadata
 */

export class ServerRegistry {
  private servers: Map<string, HTTPSServer>;  // ✅ Actual server objects (clones)!
  
  constructor() {
    this.servers = new Map();
  }
  
  /**
   * Register server by replicating it
   * Creates local clone of remote server
   */
  registerServer(scenario: Scenario<HTTPSServerModel>): HTTPSServer {
    // Create replica
    const replica = new HTTPSServer();
    replica.init(scenario);  // SAME uuid as original!
    
    // Store clone
    this.servers.set(replica.ior.uuid, replica);
    
    console.log(`✅ Server registered: ${replica.ior.uuid} (${replica.model.host}:${replica.model.port})`);
    
    return replica;
  }
  
  /**
   * Get server clone by UUID
   */
  getServer(uuid: string): HTTPSServer | null {
    return this.servers.get(uuid) || null;
  }
  
  /**
   * Get all server clones
   */
  getAllServers(): HTTPSServer[] {
    return Array.from(this.servers.values());
  }
  
  /**
   * Call method on all server clones
   * Web4 Pattern: Broadcast = call method on all clones
   */
  async broadcastMethodCall(methodName: string, ...args: any[]): Promise<void> {
    for (const server of this.servers.values()) {
      if (typeof (server as any)[methodName] === 'function') {
        await (server as any)[methodName](...args);
      }
    }
  }
  
  /**
   * Unregister server (remove clone)
   */
  unregisterServer(uuid: string): void {
    this.servers.delete(uuid);
    console.log(`❌ Server unregistered: ${uuid}`);
  }
}
```

// src/ts/layer2/PrimaryServer.ts (Conceptual - part of ONCE or HTTPSServer)
/**
 * Primary Server - Maintains clones of all client servers
 * Web4 Pattern: Registry of server replicas, not message hub
 */

export class PrimaryServer {
  private registry: ServerRegistry;
  private wsServer: WebSocketServer;
  
  constructor() {
    this.registry = new ServerRegistry();
  }
  
  /**
   * Start primary server on 42777
   */
  async start(): Promise<void> {
    // Start WebSocket server
    this.wsServer = new WebSocketServer({ port: 42777 });
    
    this.wsServer.on('connection', (ws) => {
      ws.on('message', (data) => this.handleObjectReplication(ws, data));
    });
    
    console.log('✅ Primary server started on 42777');
  }
  
  /**
   * Handle incoming object replication (no protocol parsing!)
   * Web4 Pattern: Receive scenarios, create clones
   */
  private async handleObjectReplication(ws: WebSocket, data: any): Promise<void> {
    try {
      const scenario = JSON.parse(data.toString()) as Scenario<Model>;
      
      // Determine object type from IOR
      switch (scenario.ior.component) {
        case 'HTTPSServer':
          // Client server registering itself!
          const serverReplica = this.registry.registerServer(scenario as Scenario<HTTPSServerModel>);
          
          // Replicate this server to all OTHER clients (broadcast)
          await this.replicateToAllClients(scenario, ws);
          break;
          
        case 'User':
          // User object replication
          const user = new DefaultUser();
          user.init(scenario);
          // ... handle user
          break;
          
        // ... other component types
      }
      
    } catch (error) {
      console.error('Failed to replicate object:', error);
    }
  }
  
  /**
   * Replicate object to all client servers
   * Web4 Pattern: Broadcast = replicate to all clones
   */
  private async replicateToAllClients(scenario: Scenario<Model>, excludeWs?: WebSocket): Promise<void> {
    const servers = this.registry.getAllServers();
    
    for (const server of servers) {
      // Get WebSocket connection to this server
      const ws = this.getConnectionToServer(server.ior.uuid);
      
      if (ws && ws !== excludeWs && ws.readyState === WebSocket.OPEN) {
        // Send scenario (no protocol wrapper!)
        ws.send(JSON.stringify(scenario));
      }
    }
  }
  
  /**
   * Get list of registered server scenarios (for /servers endpoint)
   */
  async getServerList(): Promise<Scenario<HTTPSServerModel>[]> {
    const servers = this.registry.getAllServers();
    
    // Hibernate all clones back to scenarios
    return Promise.all(
      servers.map(server => server.toScenario())
    );
  }

### **3. Client Server (Ports 8080+)**

```typescript
// src/ts/layer2/ClientServer.ts (Conceptual - part of HTTPSServer)
/**
 * Client Server - Replicates itself to primary, receives peer replicas
 * Web4 Pattern: Self-registration via object replication
 */

export class ClientServer extends HTTPSServer {
  private primaryConnection: WebSocket | null = null;
  private peerReplicas: Map<string, HTTPSServer>;  // ✅ Clones of peer servers!
  
  constructor() {
    super();
    this.peerReplicas = new Map();
  }
  
  /**
   * Connect to primary and replicate self
   * Web4 Pattern: Send scenario (no registration message)
   */
  async connectToPrimary(primaryHost: string, primaryPort: number): Promise<void> {
    // Connect via WebSocket
    this.primaryConnection = new WebSocket(`ws://${primaryHost}:${primaryPort}`);
    
    await new Promise<void>((resolve) => {
      this.primaryConnection!.on('open', () => {
        console.log(`✅ Connected to primary ${primaryHost}:${primaryPort}`);
        resolve();
      });
    });
    
    // Listen for peer replicas
    this.primaryConnection.on('message', (data) => this.handlePeerReplication(data));
    
    // Replicate self to primary
    await this.replicateSelfToPrimary();
  }
  
  /**
   * Replicate self to primary (no registration message!)
   * Web4 Pattern: Send own scenario
   */
  private async replicateSelfToPrimary(): Promise<void> {
    // Hibernate self to scenario
    const myScenario = await this.toScenario();
    
    // Send scenario (no protocol wrapper!)
    this.primaryConnection!.send(JSON.stringify(myScenario));
    
    console.log(`✅ Replicated self to primary: ${this.ior.uuid} (${this.model.host}:${this.model.port})`);
  }
  
  /**
   * Handle peer server replication from primary
   * Web4 Pattern: Receive peer scenarios, create clones
   */
  private async handlePeerReplication(data: any): Promise<void> {
    try {
      const scenario = JSON.parse(data.toString()) as Scenario<Model>;
      
      if (scenario.ior.component === 'HTTPSServer') {
        // Another server replicating itself!
        const peerScenario = scenario as Scenario<HTTPSServerModel>;
        
        // Skip if it's us
        if (peerScenario.ior.uuid === this.ior.uuid) {
          return;
        }
        
        // Create local clone of peer server
        const peerReplica = new HTTPSServer();
        peerReplica.init(peerScenario);  // SAME uuid as peer!
        
        // Store peer replica
        this.peerReplicas.set(peerReplica.ior.uuid, peerReplica);
        
        console.log(`✅ Peer replicated: ${peerReplica.ior.uuid} (${peerReplica.model.host}:${peerReplica.model.port})`);
      }
      
    } catch (error) {
      console.error('Failed to handle peer replication:', error);
    }
  }
  
  /**
   * Get all peer server clones
   */
  getPeerServers(): HTTPSServer[] {
    return Array.from(this.peerReplicas.values());
  }
  
  /**
   * Relay object to specific peer
   * Web4 Pattern: Get peer clone, send via its connection
   */
  async relayToPeer(peerUuid: string, scenario: Scenario<Model>): Promise<void> {
    const peer = this.peerReplicas.get(peerUuid);
    
    if (!peer) {
      throw new Error(`Peer ${peerUuid} not found in replicas`);
    }
    
    // Get connection to peer (via primary or direct)
    const ws = this.getConnectionToPeer(peer);
    
    if (ws && ws.readyState === WebSocket.OPEN) {
      // Send scenario (no protocol wrapper!)
      ws.send(JSON.stringify(scenario));
    }
  }
  
  private getConnectionToPeer(peer: HTTPSServer): WebSocket | null {
    // Implementation: Either direct connection or route via primary
    return null;  // Stub
  }
}
```

## **Comparison: Message Protocol vs Object Replication**

| **Aspect** | **Message Protocol (OLD)** | **Object Replication (NEW)** |
|-----------|---------------------------|------------------------------|
| **Registration** | `{ type: 'register', data: {...} }` | Send `Scenario<HTTPSServerModel>` |
| **Primary Storage** | `Map<UUID, ServerEntry>` metadata | `Map<UUID, HTTPSServer>` clones |
| **Broadcast** | Send message to all connections | Replicate scenario to all clones |
| **Relay** | Send message to specific connection | Send scenario to specific clone |
| **Server List** | Return metadata array | Return `Scenario[]` from clones |
| **Peer Discovery** | Receive list message | Receive peer scenarios (replicas) |
| **State Sync** | Manual state updates in messages | Automatic (clone state = real state) |

## **Key Features Preserved**

1. ✅ **Primary Server (42777)**: Maintains registry of server clones
2. ✅ **Client Registration**: Clients send their scenarios (self-replication)
3. ✅ **Server Discovery**: Primary sends peer scenarios to all clients
4. ✅ **Broadcast**: Replicate scenario to all server clones
5. ✅ **Relay**: Send scenario to specific server clone
6. ✅ **P2P**: Direct connections between replicas (optional)
7. ✅ **State Sync**: Periodic re-replication (send updated scenario)

## **WebSocket Communication Flow**

```typescript
// 1. Client Server Startup
const client = new HTTPSServer();
client.init({ port: 8080, host: 'localhost', ... });
await client.start();

// 2. Connect to Primary
await client.connectToPrimary('localhost', 42777);
// → Sends: Scenario<HTTPSServerModel> (self)

// 3. Primary Receives & Replicates
// → Creates: HTTPSServer clone in registry
// → Sends: Scenario to all OTHER clients

// 4. Other Clients Receive Peer Scenarios
// → Create: HTTPSServer clones of peers
// → Store: In local peer registry

// 5. Broadcast (e.g., browser sends User scenario)
const userScenario = await user.toScenario();
ws.send(JSON.stringify(userScenario));
// → Primary replicates to ALL server clones

// 6. Relay (send to specific peer)
const targetPeer = client.getPeerServers()[0];
await client.relayToPeer(targetPeer.ior.uuid, scenario);
// → Sends scenario to specific clone
```

## **Benefits of Object Replication Approach**

1. ✅ **TRUE OOP**: Registry contains actual objects (clones)
2. ✅ **No Protocol**: Just scenarios flowing around
3. ✅ **Same UUID**: Clones have identical UUID (identity)
4. ✅ **Method Calls**: Can call methods on clones
5. ✅ **State Sync**: Clone state reflects real server state
6. ✅ **Simplicity**: No message types, no handlers
7. ✅ **Type Safety**: Scenario<TModel> fully typed

## **Summary**

P2P multi-server works by **replicating server objects** (sending scenarios), not by sending protocol messages. The primary maintains **clones** of all client servers, and clients maintain **clones** of their peers. 

**Broadcast** = replicate to all clones  
**Relay** = send to specific clone  

**Pure OOP, no protocols!**
