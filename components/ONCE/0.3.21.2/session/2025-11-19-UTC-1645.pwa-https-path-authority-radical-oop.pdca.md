### **Key Design Decisions**

**QA Decision D1: Path Authority implementation?**
- **Option A**: Copy DefaultCLI's `calculateProjectRootInternal()` ✅
- **Option B**: Extend DefaultCLI directly
- **Option C**: Create shared PathAuthority utility class

**Decision**: **D1a - Copy pattern from DefaultCLI**
- ONCE doesn't need CLI functionality
- Pattern is simple and proven
- Calculate `projectRoot` once in `init()`
- Derive all paths from `projectRoot`

**QA Decision D2: HTTPS certificate management?**
- **Option A**: Let's Encrypt with auto-renewal ✅
- **Option B**: Manual certificates
- **Option C**: Self-signed certificates

**Decision**: **D2a - Let's Encrypt with `LetsEncryptCertificateUpdater`**
- Production-ready HTTPS
- Auto-renewal (check every 24h, renew if <30 days)
- Zero-downtime reload (swap cert without restart)
- ACME protocol compliance

**QA Decision D3: Certificate reload mechanism?**
- **Option A**: Hot reload - update TLS context without server restart ✅
- **Option B**: Restart server (downtime)
- **Option C**: Use reverse proxy (nginx)

**Decision**: **D3a - Hot reload TLS context**
- Node.js supports `setSecureContext()` on HTTPS servers
- Zero downtime
- Transparent to clients

**QA Decision D4: Route-to-file mapping?**
- **Option A**: Route stores EAMD.ucp relative path ✅
- **Option B**: Route stores absolute path
- **Option C**: Route calls handler function

**Decision**: **D4a - EAMD.ucp relative path**
- Clean separation: routing logic vs file serving
- Path Authority: Route calculates full path from model
- Easy to serialize (just string paths)

**QA Decision D5: Service Worker architecture?**
- **Option A**: Functional service worker script
- **Option B**: OOP class compiled to service worker ✅
- **Option C**: Hybrid (class generates script)

**Decision**: **D5b - OOP class compiled to service worker**
- `ONCEServiceWorker.ts` → compiled → `service-worker.js`
- Radical OOP everywhere
- Type-safe, testable

**QA Decision D6: IndexedDB caching strategy?**
- **Option A**: Cache on first access (lazy) ✅
- **Option B**: Pre-cache all files on install
- **Option C**: Network-first, cache fallback

**Decision**: **D6a - Lazy cache with offline fallback**
- Cache files as they're accessed
- Offline: serve from IndexedDB
- Background sync when online

**QA Decision D7: PWA manifest generation?**
- **Option A**: Static manifest.json
- **Option B**: Dynamic generation from scenario ✅
- **Option C**: Template with placeholders

**Decision**: **D7b - Dynamic generation**
- Manifest includes `hostname`, `uuid`, `version`
- Generated by `PWAManifestGenerator.toJSON()`
- Served at `/manifest.json` route

   import * as https from 'https';
   import * as fs from 'fs';
   import * as path from 'path';
   import { HttpRouter } from './HttpRouter.js';
   
   /**
    * HTTPS Server Manager - manages TLS certificates and HTTPS server
    * ✅ JavaBean style: private fields, public getters/setters
    * ✅ Empty constructor + init pattern
    */
   export class HTTPSServerManager {
     private model: {
       server?: https.Server;
       port: number;
       certPath: string;
       keyPath: string;
       router?: HttpRouter;
     };
     
     constructor() {
       // ✅ Empty - init in init()
       this.model = {
         port: 443,
         certPath: '',
         keyPath: ''
       };
     }
     
     /**
      * Initialize server with paths from parent model (Path Authority)
      */
     public init(certPath: string, keyPath: string, port: number): this {
       this.model.certPath = certPath;
       this.model.keyPath = keyPath;
       this.model.port = port;
       return this;
     }
     
     /**
      * Start HTTPS server with router
      */
     public async start(router: HttpRouter): Promise<void> {
       this.model.router = router;
       
       // Load initial certificate
       const options = this.loadCertificates();
       
       // Create HTTPS server
       this.model.server = https.createServer(options, (req, res) => {
         this.handleRequest(req, res);
       });
       
       return new Promise((resolve, reject) => {
         this.model.server!.listen(this.model.port, () => {
           console.log(`🔒 HTTPS server listening on port ${this.model.port}`);
           resolve();
         });
         
         this.model.server!.on('error', reject);
       });
     }
     
     /**
      * Load TLS certificates from files
      */
     private loadCertificates(): https.ServerOptions {
       return {
         key: fs.readFileSync(this.model.keyPath),
         cert: fs.readFileSync(this.model.certPath)
       };
     }
     
     /**
      * Hot reload certificates (zero downtime)
      * Called by LetsEncryptCertificateUpdater after renewal
      */
     public reloadCertificates(): void {
       if (!this.model.server) {
         throw new Error('Server not started');
       }
       
       const newOptions = this.loadCertificates();
       
       // ✅ Hot reload: update TLS context without restart
       this.model.server.setSecureContext(newOptions);
       
       console.log('🔄 TLS certificates reloaded (zero downtime)');
     }
     
     /**
      * Handle HTTP request using router
      */
     private handleRequest(req: any, res: any): void {
       if (!this.model.router) {
         res.writeHead(500);
         res.end('Router not initialized');
         return;
       }
       
       const result = this.model.router.route(req);
       
       if (result.route) {
         result.route.handle(req, res, result.match.params || {});
       } else {
         res.writeHead(404);
         res.end('Not Found');
       }
     }
     
     // ✅ JavaBean getters
     public getPort(): number {
       return this.model.port;
     }
     
     public getCertPath(): string {
       return this.model.certPath;
     }
     
     public getKeyPath(): string {
       return this.model.keyPath;
     }
   }

   import * as fs from 'fs';
   import * as path from 'path';
   import { execSync } from 'child_process';
   import { HTTPSServerManager } from './HTTPSServerManager.js';
   
   /**
    * Let's Encrypt Certificate Updater
    * Auto-renews certificates using ACME protocol (certbot)
    * ✅ Zero downtime: calls HTTPSServerManager.reloadCertificates()
    */
   export class LetsEncryptCertificateUpdater {
     private model: {
       domain: string;
       email: string;
       certsDir: string;
       checkIntervalMs: number;
       renewThresholdDays: number;
       httpsManager?: HTTPSServerManager;
       intervalId?: NodeJS.Timeout;
     };
     
     constructor() {
       // ✅ Empty
       this.model = {
         domain: '',
         email: '',
         certsDir: '',
         checkIntervalMs: 24 * 60 * 60 * 1000, // 24 hours
         renewThresholdDays: 30
       };
     }
     
     /**
      * Initialize with domain, email, paths (Path Authority)
      */
     public init(domain: string, email: string, certsDir: string): this {
       this.model.domain = domain;
       this.model.email = email;
       this.model.certsDir = certsDir;
       return this;
     }
     
     /**
      * Inject HTTPS server manager (for hot reload)
      */
     public setHTTPSManager(manager: HTTPSServerManager): this {
       this.model.httpsManager = manager;
       return this;
     }
     
     /**
      * Start auto-renewal checker (runs every 24h)
      */
     public startAutoRenewal(): void {
       console.log('🔐 Starting Let\'s Encrypt auto-renewal (checks every 24h)');
       
       // Initial check
       this.checkAndRenew();
       
       // Schedule periodic checks
       this.model.intervalId = setInterval(() => {
         this.checkAndRenew();
       }, this.model.checkIntervalMs);
     }
     
     /**
      * Stop auto-renewal
      */
     public stopAutoRenewal(): void {
       if (this.model.intervalId) {
         clearInterval(this.model.intervalId);
         this.model.intervalId = undefined;
       }
     }
     
     /**
      * Check certificate expiration and renew if needed
      */
     private async checkAndRenew(): Promise<void> {
       try {
         const certPath = path.join(this.model.certsDir, 'fullchain.pem');
         
         if (!fs.existsSync(certPath)) {
           console.log('📜 No certificate found, requesting new one...');
           await this.requestNewCertificate();
           return;
         }
         
         const daysUntilExpiry = this.getDaysUntilExpiry(certPath);
         console.log(`📅 Certificate expires in ${daysUntilExpiry} days`);
         
         if (daysUntilExpiry <= this.model.renewThresholdDays) {
           console.log('🔄 Renewing certificate...');
           await this.renewCertificate();
           
           // ✅ Hot reload certificates (zero downtime)
           if (this.model.httpsManager) {
             this.model.httpsManager.reloadCertificates();
           }
         }
       } catch (error) {
         console.error('❌ Certificate renewal check failed:', error);
       }
     }
     
     /**
      * Get days until certificate expiry
      */
     private getDaysUntilExpiry(certPath: string): number {
       const cert = fs.readFileSync(certPath, 'utf8');
       
       // Parse certificate using openssl
       const output = execSync(
         `openssl x509 -enddate -noout`,
         { input: cert, encoding: 'utf8' }
       );
       
       // Output: "notAfter=Dec 18 12:00:00 2024 GMT"
       const match = output.match(/notAfter=(.+)/);
       if (!match) {
         throw new Error('Cannot parse certificate expiry date');
       }
       
       const expiryDate = new Date(match[1]);
       const now = new Date();
       const diffMs = expiryDate.getTime() - now.getTime();
       const diffDays = Math.floor(diffMs / (1000 * 60 * 60 * 24));
       
       return diffDays;
     }
     
     /**
      * Request new certificate from Let's Encrypt (ACME)
      */
     private async requestNewCertificate(): Promise<void> {
       const cmd = `certbot certonly --standalone \
         --non-interactive \
         --agree-tos \
         --email ${this.model.email} \
         --domains ${this.model.domain} \
         --cert-path ${this.model.certsDir}`;
       
       execSync(cmd, { stdio: 'inherit' });
       console.log('✅ New certificate obtained');
     }
     
     /**
      * Renew existing certificate
      */
     private async renewCertificate(): Promise<void> {
       const cmd = `certbot renew --cert-name ${this.model.domain}`;
       execSync(cmd, { stdio: 'inherit' });
       console.log('✅ Certificate renewed');
     }
     
     // ✅ JavaBean getters/setters
     public getDomain(): string {
       return this.model.domain;
     }
     
     public setCheckInterval(ms: number): this {
       this.model.checkIntervalMs = ms;
       return this;
     }
     
     public setRenewalThreshold(days: number): this {
       this.model.renewThresholdDays = days;
       return this;
     }
   }

     private async serveFile(res: any): Promise<void> {
       if (!this.model.eamdBasePath) {
         res.writeHead(500);
         res.end('EAMD base path not set');
         return;
       }
       
       // ✅ Path Authority: construct full path from base
       const fullPath = path.join(this.model.eamdBasePath, this.model.target);
       
       // Security: prevent directory traversal
       if (!fullPath.startsWith(this.model.eamdBasePath)) {
         res.writeHead(403);
         res.end('Forbidden');
         return;
       }
       
       if (!fs.existsSync(fullPath)) {
         res.writeHead(404);
         res.end('Not Found');
         return;
       }
       
       const content = fs.readFileSync(fullPath);
       const contentType = mime.lookup(fullPath) || 'application/octet-stream';
       
       res.writeHead(200, { 'Content-Type': contentType });
       res.end(content);
     }
     
     // ✅ JavaBean getters
     public getPath(): string {
       return this.model.path;
     }
     
     public getMethod(): HttpMethod {
       return this.model.method;
     }
     
     public getTarget(): string {
       return this.model.target;
     }
   }

   export class ONCEServiceWorker {
     private model: {
       cacheName: string;
       version: string;
       cachedUrls: string[];
     };
     
     constructor() {
       this.model = {
         cacheName: 'once-pwa-cache',
         version: '0.3.21.2',
         cachedUrls: []
       };
     }
     
     /**
      * Install event - cache essential files
      */
     public async onInstall(event: ExtendableEvent): Promise<void> {
       console.log('[Service Worker] Installing...');
       
       // ✅ Cache essential files for offline
       const essentialUrls = [
         '/EAMD.ucp/components/ONCE/0.3.21.2/dist/ts/layer2/DefaultONCE.js',
         '/EAMD.ucp/components/ONCE/0.3.21.2/dist/ts/layer2/ONCEBrowserClient.js',
         '/EAMD.ucp/components/ONCE/0.3.21.2/src/view/html/demo.html',
         '/manifest.json',
         '/offline.html'
       ];
       
       await this.cacheUrls(essentialUrls);
     }
     
     /**
      * Activate event - clean old caches
      */
     public async onActivate(event: ExtendableEvent): Promise<void> {
       console.log('[Service Worker] Activating...');
       
       // Clean old cache versions
       const cacheKeys = await caches.keys();
       const oldCaches = cacheKeys.filter(key => 
         key.startsWith('once-pwa-cache') && key !== this.getCacheName()
       );
       
       await Promise.all(oldCaches.map(key => caches.delete(key)));
     }
     
     /**
      * Fetch event - serve from cache or network
      */
     public async onFetch(event: FetchEvent): Promise<Response> {
       const request = event.request;
       
       // Strategy: Cache-first for static assets, network-first for API calls
       if (this.isStaticAsset(request.url)) {
         return await this.cacheFirst(request);
       } else {
         return await this.networkFirst(request);
       }
     }
     
     /**
      * Cache-first strategy
      */
     private async cacheFirst(request: Request): Promise<Response> {
       const cached = await caches.match(request);
       if (cached) {
         return cached;
       }
       
       try {
         const response = await fetch(request);
         await this.cacheResponse(request, response.clone());
         return response;
       } catch (error) {
         // Offline fallback
         return new Response('Offline', { status: 503 });
       }
     }
     
     /**
      * Network-first strategy
      */
     private async networkFirst(request: Request): Promise<Response> {
       try {
         const response = await fetch(request);
         await this.cacheResponse(request, response.clone());
         return response;
       } catch (error) {
         const cached = await caches.match(request);
         if (cached) {
           return cached;
         }
         throw error;
       }
     }
     
     /**
      * Cache URLs
      */
     private async cacheUrls(urls: string[]): Promise<void> {
       const cache = await caches.open(this.getCacheName());
       await cache.addAll(urls);
       this.model.cachedUrls.push(...urls);
     }
     
     /**
      * Cache single response
      */
     private async cacheResponse(request: Request, response: Response): Promise<void> {
       const cache = await caches.open(this.getCacheName());
       await cache.put(request, response);
     }
     
     /**
      * Check if URL is static asset
      */
     private isStaticAsset(url: string): boolean {
       return /\.(js|css|html|png|jpg|svg|woff2?)$/.test(url);
     }
     
     /**
      * Get versioned cache name
      */
     private getCacheName(): string {
       return `${this.model.cacheName}-v${this.model.version}`;
     }
   }
   
   // ✅ Self-executing: register event handlers
   const worker = new ONCEServiceWorker();
   
   self.addEventListener('install', (event: any) => {
     event.waitUntil(worker.onInstall(event));
   });
   
   self.addEventListener('activate', (event: any) => {
     event.waitUntil(worker.onActivate(event));
   });
   
   self.addEventListener('fetch', (event: any) => {
     event.respondWith(worker.onFetch(event));
