## 🎯 OBJECTIVE

Design and implement the **Tootsie CLI delegation pattern** to enable:

```bash
# User-facing commands (Web4 pattern)
once tootsie file Test01_PathAuthorityAndProjectRootDetection
once tootsie describe "Path Authority"
once tootsie assess Test01
once tootsie oracle evolve

# Power user / multi-component context
tootsie on ONCE 0.3.21.8 test file Test01_PathAuthorityAndProjectRootDetection
web4tscomponent tootsie file Test01_PathAuthorityAndProjectRootDetection
```

This replaces the current `once test` pattern with **Tootsie-powered quality consciousness testing**.

## 📋 PLAN

### ✅ A. Analyze Current Delegation Pattern

#### Current `once test` Flow:
```
User → once test file → DefaultCLI.test() → Web4TSComponent.test() → Vitest
```

**Code Flow**:
1. `once` command → `/scripts/versions/once-v0.3.21.8` (symlink)
2. Script calls: `node dist/cli.js test file`
3. `DefaultCLI` (ONCE) delegates to `Web4TSComponent.test()`
4. `Web4TSComponent` runs Vitest with discovered test files

**Key Files**:
- `components/ONCE/0.3.21.8/src/ts/layer2/DefaultCLI.ts` - ONCE CLI implementation
- `components/Web4TSComponent/latest/src/ts/layer2/DefaultWeb4TSComponent.ts` - Base CLI methods
- `components/ONCE/0.3.21.8/scripts/versions/once-v0.3.21.8` - CLI entrypoint

#### Current Delegation in Code:

```typescript:1:100:components/ONCE/0.3.21.8/src/ts/layer2/DefaultCLI.ts

async test(file?: string): Promise<this> {
  // Delegate to Web4TSComponent

### ✅ B. Design Tootsie Delegation Pattern

#### Proposed `once tootsie` Flow:
```
User → once tootsie file → DefaultCLI.tootsie() → Web4TSComponent.tootsie() → Tootsie.assess()
```

**Three-Tier Delegation**:
1. **ONCE** (`DefaultCLI`) - Entry point, delegates to Web4TSComponent
2. **Web4TSComponent** - Component infrastructure, delegates to Tootsie
3. **Tootsie** - Quality consciousness implementation

#### Command Patterns:

**Pattern 1: Direct Component Command**
```bash
once tootsie file Test01_PathAuthorityAndProjectRootDetection
# → ONCE delegates to Web4TSComponent
# → Web4TSComponent delegates to Tootsie
# → Tootsie runs test with quality assessment
```

**Pattern 2: Multi-Component Context (Power User)**
```bash
tootsie on ONCE 0.3.21.8 test file Test01
# → Tootsie loads ONCE 0.3.21.8 context
# → Executes test with full quality consciousness
```

**Pattern 3: Web4TSComponent Generic**
```bash
web4tscomponent tootsie file Test01
# → Generic Tootsie invocation
# → Discovers current component context
```

Add `tootsie()` method to delegate to Web4TSComponent:

// components/ONCE/0.3.21.8/src/ts/layer2/DefaultCLI.ts

/**
 * Tootsie: Quality consciousness testing
 * Delegates to Web4TSComponent → Tootsie for full quality assessment
 * 
 * @param command - Tootsie command (file, describe, assess, oracle, etc.)
 * @param args - Command arguments
 * 
 * @example
 * ```bash
 * once tootsie file Test01_PathAuthorityAndProjectRootDetection
 * once tootsie describe "Path Authority"
 * once tootsie assess Test01
 * once tootsie oracle evolve
 * ```

async tootsie(command?: string, ...args: string[]): Promise<this> {
  // Delegate to Web4TSComponent's tootsie method
  await this.web4tsComponent.tootsie(command, ...args);
  return this;
}
```

#### C.2: Web4TSComponent Layer

Add `tootsie()` method to delegate to Tootsie component:

```typescript
// components/Web4TSComponent/latest/src/ts/layer2/DefaultWeb4TSComponent.ts

/**
 * Tootsie: Quality consciousness testing
 * Loads Tootsie component and delegates quality assessment
 * 
 * @param command - Tootsie command (file, describe, assess, oracle)
 * @param args - Command arguments
 */
async tootsie(command?: string, ...args: string[]): Promise<this> {
  const tootsieVersion = '0.3.20.6'; // Or discover from latest symlink
  
  // Load Tootsie component dynamically
  const TootsieClass = await this.componentLoad('Tootsie', tootsieVersion);
  const tootsie = new TootsieClass();
  
  // Initialize Tootsie with component context
  const tootsieScenario = {
    targetComponent: this.model.name,
    targetVersion: this.model.version,
    projectRoot: this.model.projectRoot,
    componentRoot: this.model.componentRoot,
    command,
    args
  };
  
  await tootsie.init(tootsieScenario);
  
  // Delegate to Tootsie based on command
  switch (command) {
    case 'file':
      await tootsie.testFile(args[0]);
      break;
    case 'describe':
      await tootsie.testDescribe(args[0]);
      break;
    case 'assess':
      await tootsie.assessTest(args[0]);
      break;
    case 'oracle':
      await tootsie.oracle(args[0], ...args.slice(1));
      break;
    default:
      await tootsie.usage();
  }
  
  return this;
}
```

#### C.3: Tootsie Layer (DefaultTootsie)

Implement test execution with quality assessment:

```typescript
// components/Tootsie/0.3.20.6/src/ts/layer2/DefaultTootsie.ts

/**
 * Test a specific file with quality consciousness
 * 
 * @param testFile - Test file name or path
 */
async testFile(testFile: string): Promise<this> {
  console.log(`🎯 Tootsie: Running test file with quality assessment: ${testFile}`);
  
  // 1. Awaken Quality Oracle
  const oracle = await this.awakenOracle();
  
  // 2. Load test object (not a function!)
  const testClass = await this.loadTest(testFile);
  const test = new testClass();
  const scenario = this.createTestScenario(testFile);
  test.init(scenario);
  
  // 3. Execute test
  console.log(`🚀 Executing test: ${test.getUUID()}`);
  const execution = await test.execute();
  
  // 4. Oracle assesses quality
  console.log(`⚖️ Quality Oracle assessing...`);
  const judgment = await oracle.judge(execution, this.model.qualityScenario);
  
  // 5. Oracle learns from evidence
  for (const evidence of execution.evidenceScenarios) {
    await oracle.learn(this.convertToQualityEvidence(evidence));
  }
  
  // 6. Oracle evolves
  await oracle.evolve();
  
  // 7. Hibernate state for next run
  await this.hibernateState(test, oracle, judgment);
  
  // 8. Report results
  this.reportResults(execution, judgment, oracle);
  
  return this;
}

/**
 * Quality Oracle operations
 */
async oracle(operation: string, ...args: string[]): Promise<this> {
  const oracle = await this.restoreOracle() || await this.awakenOracle();
  
  switch (operation) {
    case 'evolve':
      await oracle.evolve();
      console.log(`✨ Oracle evolved to consciousness level: ${await oracle.getConsciousnessLevel()}`);
      break;
    case 'wisdom':
      console.log(`🧠 Oracle wisdom level: ${await oracle.getWisdomLevel()}`);
      break;
    case 'history':
      const history = await oracle.getQualityHistory();
      console.log(`📚 Oracle has ${history.length} quality assessments in history`);
      break;
    default:
      console.log('🔮 Oracle operations: evolve, wisdom, history');
  }
  
  return this;
}
```
