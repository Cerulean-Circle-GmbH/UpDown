## **🎯 ACT**

**Fixing What Was Found Broken in CHECK:**

**Bug 1: Endpoints Section Not Rendering**
- **Problem:** `OncePeerDefaultView` receives `BrowserOnceModel` instead of `ServerDefaultModel`
- **Root Cause:** Commit `d8e3552` changed main route from `serveDemoLit()` to `serveOnceApp()`. `serveDemoLit()` served `demo-lit.html` which populated model. `serveOnceApp()` serves `once.html` SPA which uses `UcpRouter`, and `UcpRouter` sets `this.model = this.component.browserModel` (orchestrator line 79), but `BrowserOnceModel` doesn't have server data.
- **Evidence:** 
  - In `d729313`: `homeRoute.setProvider(() => this.component!.serveDemoLit());` - worked
  - After: `homeRoute.setProvider(() => this.component!.serveOnceApp());` - broken
  - `serveDefaultView()` (lines 2298-2364) correctly populates model but isn't used for main route
- **Fix:** When `UcpRouter` creates `once-peer-default-view` for `/` route, fetch server model (like `serveDefaultView()` does) and populate `ServerDefaultModel`
- **Location:** `UcpRouter.viewCreate()` or `BrowserOnceOrchestrator.routesRegister()` needs to fetch server model when creating `once-peer-default-view`

**Bug 2: Shows "Client Server" Instead of "Primary Server"**
- **Problem:** `isPrimaryServer` is `undefined` in `BrowserOnceModel`, defaults to `false`
- **Root Cause:** Same as Bug 1 - wrong model type passed to view
- **Evidence:** View shows "🔵 Client Server" even though server is primary (port 42777)
- **Fix:** Fetch server status and set `isPrimaryServer: true` in model when server is primary
- **Location:** Need to check `ServerHierarchyManager.isPrimaryServer()` and populate model

**Bug 3: IOR Routing Errors with Invalid UUID**
- **Problem:** `[IORMethodRouter] Routing failed: Error: Component not found: ONCE/0.3.21.8/invalid-uuid`
- **Root Cause:** Something is calling IORs with invalid UUIDs
- **Evidence:** Terminal shows IOR routing errors during page load
- **Fix:** Investigate what's calling invalid IORs - likely browser trying to call methods before UUID is known
- **Location:** Check browser initialization code, WebSocket handlers
