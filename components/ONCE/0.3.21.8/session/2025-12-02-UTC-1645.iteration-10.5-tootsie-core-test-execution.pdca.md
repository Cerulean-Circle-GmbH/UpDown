### Task 1: Implement Test File Discovery

**Goal**: Find test files by number/name/token in `/test/tootsie/`

**Steps**:
1. Add `discoverTestFile()` method to Web4TSComponent
2. Support file number: `1` → `Test01_*.ts`
3. Support full name: `Test01_PathAuthorityAndProjectRootDetection`
4. Support hierarchical tokens (future): `1a`, `1a2`
5. Return absolute path to test file

**Expected Output**:
```typescript
discoverTestFile('1') 
  → '.../test/tootsie/Test01_PathAuthorityAndProjectRootDetection.ts'
```

### Task 3: Implement Test Execution Flow

**Goal**: Execute tests through Tootsie with evidence collection

**Steps**:
1. Discover test file based on scope/references
2. Load test class dynamically (`import()`)
3. Instantiate test case with scenario
4. Execute test methods (respecting describe/itCase filters)
5. Collect evidence at each step
6. Return test results

**Flow**:
```
Web4TSComponent.tootsie('file', '1')
  ↓
discoverTestFile('1') → Test01_PathAuthority...ts
  ↓
Load Tootsie component
  ↓
tootsie.executeTestFile(testPath, scenario)
  ↓
Test runs, evidence collected
  ↓
Return results to Web4TSComponent
```

**Goal**: Oracle judges test outcomes and learns

**Steps**:

4. Oracle updates wisdom based on judgment
5. Display oracle assessment

```
🧠 Quality Oracle Assessment:

   Wisdom Gained: "Path authority correctly detects test isolation"
   Evidence Quality: EXCELLENT (complete trail)
```

### Task 5: Evidence & State Hibernation

**Goal**: Persist test state and evidence to `test/data/.tootsie/`

**Directory Structure**:
```
test/data/.tootsie/
├── evidence/
│   └── test01/
│       ├── input.json          # Initial scenario
│       ├── step-001.json       # Each assertion
│       ├── step-002.json
│       └── output.json         # Final state
├── state/
│   └── test01.json             # Hibernated test object
└── oracle/
    └── wisdom.json             # Accumulated learning
```

**Steps**:
1. After test execution, save evidence trail
2. Hibernate test object to `state/test01.json`
3. Update oracle wisdom in `oracle/wisdom.json`
4. All paths scoped to `test/data/.tootsie/`

---

### Task 6: Implement Scope Variations

**Goal**: Support all scope types (file, describe, itCase)

**Patterns**:
```bash
once tootsie file 1           # Run all tests in Test01
once tootsie describe 1a      # Run describe block 1a in Test01
once tootsie itCase 1a2       # Run specific test 1a2 in Test01
```

**Implementation**:
- Parse hierarchical reference (e.g., `1a2` → file=1, describe=a, itCase=2)
- Pass filters to Tootsie execution
- Tootsie respects filters when running tests
