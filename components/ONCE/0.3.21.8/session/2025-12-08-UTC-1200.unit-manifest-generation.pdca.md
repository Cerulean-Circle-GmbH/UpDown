## 📋 PLAN: Phased Unit Integration

### Architecture: Unit Discovery & Manifest

```
┌─────────────────────────────────────────────────────────────────┐
│                    UnitDiscoveryService (NEW)                    │
│                    Layer 2 - Domain Logic                        │
├─────────────────────────────────────────────────────────────────┤
│  unitsDiscover(componentRoot: string): UnitDefinition[]         │
│  unitCreate(definition: UnitDefinition): Scenario               │
│  manifestGenerate(units: Scenario[]): ComponentManifest         │
│  manifestSave(manifest: ComponentManifest): void                │
├─────────────────────────────────────────────────────────────────┤
│                           ↓                                      │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │               ScenarioService (existing)                     ││
│  │  scenarioCreate(), scenarioSave(), scenarioLoad()            ││
│  └─────────────────────────────────────────────────────────────┘│
│                           ↓                                      │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │               UcpStorage (existing)                          ││
│  │  Index-based storage with symlinks                           ││
│  └─────────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────────┘
```

### Unit Types & TypeM3 Classification

| File Type | TypeM3 | Example |
