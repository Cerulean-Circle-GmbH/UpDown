**Return Type Contract** (Web4 Radical OOP):
```typescript
// ✅ ONLY two return types allowed:
// 1. this (method chaining)
// 2. Scenario<T> (typed object state)

// ✅ Example: Method chaining
public registerRoute(route: Route): this {
    this.routes.set(route.model.path, route);
    return this; // ← Method chaining

// ✅ Example: Scenario return
public async handle(req: Request): Promise<Scenario<HealthModel>> {
    const health = this.component.getHealth();
    return health.toScenario(); // ← Typed Scenario
}

// ❌ NEVER: Naked untyped JSON

}
```

**RESTLoader Purpose** (clarification):
- **NOT for our routes** (we use Scenario routes)
- **FOR ONCE to call external REST servers** via `ior:rest:https://external-api.com/endpoint`
- Example: ONCE needs to call a legacy REST API → Use RESTLoader
- Our internal routes → Use ScenarioRoute (return `Scenario<T>`)

**Route Type Clarification**:
- `HTMLRoute`: Returns HTML pages (string content)
- `RESTRoute`: For calling **external** REST APIs (via RESTLoader)
- `ScenarioRoute`: Returns `Scenario<T>` (our primary route type)
- `IORRoute`: Parses IOR URLs, delegates to ScenarioRoute
- ❌ `JSONRoute`: **DELETED** - No naked JSON ever
