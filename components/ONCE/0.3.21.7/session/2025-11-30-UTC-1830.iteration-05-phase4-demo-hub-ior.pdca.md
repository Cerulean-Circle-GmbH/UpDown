## **📋 P - PLAN**

### **1. Problem Statement**

`demo-hub.html` lost its lifecycle management capabilities during the migration to unified kernel architecture (Iteration 1.6+). The old version (0.3.20.5) had functional REST endpoints (`/start-server`, `/stop-server`, `/shutdown-all`, `/discover-servers`) which violate Web4 Principle 12 (IOR-based Method Invocation).

**Current State (0.3.21.7)**:
- ✅ Displays peer scenarios from kernel model
- ✅ Real-time updates via WebSocket
- ❌ **NO lifecycle management** (start/stop servers)
- ❌ No buttons for server control

**Old State (0.3.20.5)**:
- ✅ Full lifecycle management
- ✅ Start next server button
- ✅ Stop individual servers
- ✅ Discover servers (housekeeping)
- ✅ Graceful shutdown all
- ❌ **Used functional REST endpoints** (violates Web4)

### **2. Target Situation (TO-BE)**

Restore demo-hub lifecycle management using **IOR-based method invocation** (Web4 Principle 12):

1. ✅ **Start Next Server**: `ior:scenario:REST:https://{primaryHost}:42777/ONCE/0.3.21.7/{primaryUUID}/startClientServer`
2. ✅ **Stop Server**: `ior:scenario:REST:https://{primaryHost}:42777/ONCE/0.3.21.7/{serverUUID}/stopServer`
3. ✅ **Discover Servers**: `ior:scenario:REST:https://{primaryHost}:42777/ONCE/0.3.21.7/{primaryUUID}/discoverServers`
4. ✅ **Shutdown All**: `ior:scenario:REST:https://{primaryHost}:42777/ONCE/0.3.21.7/{primaryUUID}/shutdownAll`

**Key Differences from 0.3.20.5**:
- **IOR URLs** instead of functional REST paths (`/start-server` → `/ONCE/0.3.21.7/{uuid}/startClientServer`)
- **Method calls** on component instances (not stateless functions)
- **Scenario-based responses** (not JSON status messages)
- **UUID-based routing** (targets specific ONCE instance)

### **Phase 4.2: Implement IOR-based Functions** (60 min)

```javascript
// Get primary server UUID from kernel
const primaryUUID = kernel.model.uuid;

// Start next client server
async function startNextServer() {
    const ior = `http://${peerHost}/ONCE/0.3.21.7/${primaryUUID}/startClientServer`;
    const response = await fetch(ior, { method: 'POST' });
    const scenario = await response.json();
    addLog('lifecycle', `✅ Server starting: ${scenario.ior.uuid}`);
    updatePeerDisplay(); // Refresh
}

// Stop specific server
async function stopServer(uuid) {
    const ior = `http://${peerHost}/ONCE/0.3.21.7/${uuid}/stopServer`;
    const response = await fetch(ior, { method: 'POST' });
    addLog('lifecycle', `🛑 Stopped server: ${uuid.substring(0, 8)}`);
    updatePeerDisplay(); // Refresh
}

// Discover servers (housekeeping)
async function discoverServers() {
    const ior = `http://${peerHost}/ONCE/0.3.21.7/${primaryUUID}/discoverServers`;
    const response = await fetch(ior, { method: 'POST' });
    const result = await response.json();
    addLog('lifecycle', `🔍 Discovery: ${result.discovered} found, ${result.deleted} removed`);
    updatePeerDisplay(); // Refresh
}

// Shutdown all servers
async function shutdownAll() {
    if (!confirm('Shutdown ALL servers?')) return;
    const ior = `http://${peerHost}/ONCE/0.3.21.7/${primaryUUID}/shutdownAll`;
    await fetch(ior, { method: 'POST' });
    addLog('lifecycle', '🛑 Shutdown initiated');
    setTimeout(() => window.location.reload(), 3000);
}
```
