## **P - PLAN**

### **1. Problem Statement**

**Current Situation**:
- IORs are just identifiers (protocol://host:port/component/version/uuid)
- No way to invoke methods via IOR
- HTTP endpoints are hardcoded routes (not generic)
- Query parameters not mapped to method calls
- No generic "action endpoint" for object method invocation

**Example of Current Limitation**:
```typescript
// ❌ CURRENT: Hardcoded routes
app.get('/health', (req, res) => {
    res.json(once.getHealth());
});

app.get('/servers', (req, res) => {
    res.json(once.getServers());
});

// Want: Generic method invocation
// GET /ONCE/getHealth → calls global.ONCE.getHealth()
// GET /ONCE/getServers → calls global.ONCE.getServers()
// GET /ONCE/startServer?port=8080 → calls global.ONCE.startServer() with model.port=8080
```

**Why This Matters**:
- IOR should be executable (not just a reference)
- Like CLI: command → method mapping
- Like REST: URL → resource method
- Web4 Principle 12: IOR-based Method Invocation

```typescript
class DefaultONCE {
    async handleMethodInvocation(methodName: string, params: Record<string, any>) {
        // Map method name to actual method
        if (typeof this[methodName] === 'function') {
            // Set params on model (temporary)
            Object.assign(this.model, params);
            
            // Call method
            const result = await this[methodName]();
            
            return result;
        }
        throw new Error(`Method ${methodName} not found`);
    }
}
```

**Architecture**:
```
HTTP Request                    Method Invocation
  ↓                                ↓
Parse IOR URL                  Extract method name
  ↓                                ↓
/Component/method?params       Parse query params
  ↓                                ↓
Find component instance        Set params on model
(global.ONCE, etc.)               ↓
  ↓                            Call method()
Invoke method                     ↓
  ↓                            Return result
Return JSON result
```

#### **Task 1: Add Generic Method Invocation to DefaultONCE**

**File**: `DefaultONCE.ts`

**New Method**:
```typescript
/**
 * Handle method invocation via IOR
 * Maps URL endpoint to method call (like CLI does)
 * @pdca 2025-11-22-UTC-1730.iteration-01.6.5-ior-method-invocation.pdca.md
 */
async invokeMethod(methodName: string, params: Record<string, any> = {}): Promise<any> {
    // Check if method exists
    if (typeof (this as any)[methodName] !== 'function') {
        throw new Error(`Method '${methodName}' not found on ONCE`);
    }
    
    // Temporarily set params on model (for parameterless methods)
    const originalValues: Record<string, any> = {};
    for (const [key, value] of Object.entries(params)) {
        if (key in this.model) {
            originalValues[key] = (this.model as any)[key];
        }
        (this.model as any)[key] = value;
    }
    
    try {
        // Call method
        const result = await (this as any)[methodName]();
        return result;
    } finally {
        // Restore original values
        for (const [key, value] of Object.entries(originalValues)) {
            (this.model as any)[key] = value;
        }
    }
}
```

```typescript
private async handleHttpRequest(req: any, res: any): Promise<void> {
    const url = new URL(req.url!, `http://${req.headers.host}`);
    const path = url.pathname;
    
    // ✅ NEW: Generic method invocation endpoint
    // Pattern: /ONCE/{method} or /ONCE/{version}/{uuid}/{method}
    const methodMatch = path.match(/^\/ONCE\/(?:[\d.]+\/[\w-]+\/)?([\w]+)$/);
    if (methodMatch) {
        const methodName = methodMatch[1];
        
        // Parse query parameters
        const params: Record<string, any> = {};
        url.searchParams.forEach((value, key) => {
            params[key] = value;
        });
        
        try {
            // Delegate to DefaultONCE
            const result = await (this.component as any).invokeMethod(methodName, params);
            
            res.writeHead(200, { 'Content-Type': 'application/json' });
            res.end(JSON.stringify(result));
            return;
        } catch (error: any) {
            res.writeHead(404, { 'Content-Type': 'application/json' });
            res.end(JSON.stringify({ 
                error: error.message,
                method: methodName,
                availableMethods: this.getAvailableMethods()
            }));
            return;
        }
    }
    
    // Existing hardcoded routes (for backward compatibility)
    if (path === '/health') {
        // ...
    }
}

private getAvailableMethods(): string[] {
    // Return list of public methods
    return [
        'getHealth',
        'getServers',
        'info',
        'startServer',
        'stopServer',
        // ... etc
    ];
}
```
