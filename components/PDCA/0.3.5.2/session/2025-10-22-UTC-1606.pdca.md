**CORRECT Format (PDCA Files):**
```markdown

```
- Display: `§/path/from/root` - Shows absolute path for user clarity
- Link: `../../relative/from/document` - Relative path for local navigation
- Deeper folders need more `../` to navigate up

**INCORRECT Understanding (What I Implemented):**
```markdown

```
- I made BOTH parts absolute paths
- This breaks local markdown navigation

**What I Broke:**
1. check3c - now requires both parts match (wrong)
