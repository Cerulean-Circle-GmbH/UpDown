# Research: updateComponentDescriptor Command

**Date**: 2025-11-27 UTC 17:46
**Task**: Create command to generate component.json for existing components without one

---

## Problem

**ONCE 0.3.21.7** doesn't have `ONCE.component.json` because it was created before the scenario generation feature existed.

**Command Needed**: `web4tscomponent on ONCE latest updateComponentDescriptor`

This should:
1. Load the ONCE component
2. Call its `toScenario()` method
3. Write the scenario to `scenarios/Web4TSComponent/0.3.20.6/{uuid}.scenario.json`
4. Create symlink `ONCE.component.json → ../../scenarios/.../`

---

## Implementation Location

**File**: `src/ts/layer2/DefaultWeb4TSComponent.ts`

**Method to Add**:
```typescript
/**
 * Generate component.json scenario file for existing components
 * Used to create scenario files for components that were created before this feature existed
 * 
 * Usage: web4tscomponent on ONCE latest updateComponentDescriptor
 * 
 * Web4 Principle: "Everything is a Scenario" - all components need scenario files
 * 
 * @pdca 2025-11-27-UTC-1724.migrate-component-scenario-creation.pdca.md
 * @cliSyntax
 */
async updateComponentDescriptor(): Promise<this> {
  this.printQuickHeader();
  
  // Must have context loaded via on()
  if (!this.model.context) {
    console.error('❌ Error: No component context loaded');
    console.log('   Usage: web4tscomponent on <Component> <Version> updateComponentDescriptor');
    throw new Error('updateComponentDescriptor requires context from on() command');
  }
  
  const targetComponent = this.model.context;
  const componentName = targetComponent.model.component;
  const componentVersion = targetComponent.model.version.toString();
  const componentRoot = this.model.targetComponentRoot!;
  
  console.log(`📄 Generating component descriptor for ${componentName} ${componentVersion}...`);
  
  // Call the same createComponentScenario method
  await this.createComponentScenario(componentName, componentVersion, componentRoot);
  
  console.log(`✅ Component descriptor created successfully`);
  
  return this;
}
```
