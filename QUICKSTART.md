# UpDown - Quick Start Guide 🚀

## Instant Play (3 commands)

```bash
npm start          # Starts server + opens browser
# Play the game...
npm run stop       # Stops server gracefully
```

That's it! 🎉

## In-Game Controls

Once the game loads in your browser:

### Start Playing
- Click **"Start Game"** button, OR
- Press **`U`** key (Game Up)

### During Gameplay
- **⬆ Higher** - Next card will be higher
- **⬇ Lower** - Next card will be lower  
- **= Equal** - Next card will be equal (risky, 50 pts!)

### Stop Early
- Press **`D`** key (Game Down)
