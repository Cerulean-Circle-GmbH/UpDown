# Iteration 2 Task 8 | PO: Enforce Radical OOP, Protocol-less API Design

**Date:** 2025-07-22

## Task Description
- Consolidate and formalize the radical OOP, protocol-less approach for all core API and model design.
- All actions (e.g., join, leave, playCard) must be implemented as methods on the relevant model classes (e.g., Lobby.join(), Player.playCard(), Scenario.send()).
- Remove explicit protocol-style function calls from the API spec and replace them with object-oriented method calls.
- Document this as a first principle for all future development.
- Educate all developers to follow this guideline.
- Update the API/model spec and developer documentation accordingly.

## Acceptance Criteria
- docs/api-and-model-spec.md and developer documentation reflect the OOP, protocol-less approach.
- All future tasks and code follow this principle.
