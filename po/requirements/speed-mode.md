# Speed Modus 
•	Singleplayer: Nur ein Spieler gegen das System.
•	Deckgröße: Unlimitiert – nach jedem Zug wird das Deck „neu gemischt“, der Spieler erhält jedes Mal zufällig eine Karte aus allen 52 Karten, egal wie oft bereits gespielt.
•	Deckverbrauch: Kein Verbrauch, alle Karten stehen immer wieder zur Verfügung – jede Runde ist komplett unabhängig.
•	Streaks: Spieler kann beliebig viele aufeinanderfolgende richtige Tipps (z.B. 20er Streak) erreichen, ohne dass das Deck je „leer“ wird oder die Chancen sinken.
•	Zielgruppe: Einzelspieler, Fokus auf schnelle, kontinuierliche Entscheidungsrunden, keine Einschränkung durch Deckgröße oder -verbrauch.
 
# Kernunterschied:
Im Classic Modus wird mit einem echten, endlichen Kartendeck gespielt, das sich durch Ziehen verbraucht. Im Speed Modus gibt es für den Spieler unbegrenzte, zufällig gezogene Karten, sodass er seine Streaks beliebig fortsetzen kann, ohne Einschränkungen durch die verbleibenden Karten.
